Prüfung UX-UI Design / HTML / CSS

Prüfungsregeln

  1. Dir Prüfung dauert 3 Tage
  2. Alles, was in Regeln nicht verboten ist, ist erlaubt.
  3. Die Aufgabe ist als Challenge konzipiert. In echten Leben, dauert es wirklich Monaten... Wir werden keine perfekte Lösung bekommen. Es ist wichtig die Zwischenergebnisse feiern.

Aufgabestellung

Design und implementiere eine neue Webseite, für die Digital Campus Vorarlberg

Regel: Bilder, Farben und Schriftarten aus der offiziellen Seite sind TABU.

Das Endergbenis besteht aus:

  1. Hauptseite, mit Liste der Kursen
  2. Seite für Kursdetails
  3. Bonusaufgabe: Seite für Kursanmeldung

Teilaufgaben

Recherche

Recherche die aktuelle Seite und entwickle eine Idee, was du machen möchtest. (0 Pts)

Erstelle eine Dummy Seite, um die Wirkung der Kombination von Farben, Fonts selbst überprüfen kannst. (3 Pts)

Erstelle Boxes mit unterschiedliche Hintergrundfarben, Schriftfarben, Schriftgrößen

Erkläre welche Emotion/Qualität du mit der Farbe / Schriftart in Ausdruck bringen willst

Erstelle ein CTA Button, und prüfe, wie es auf die unterschiedlichen Hintergrundfarben wirkt

Erstelle ein Layout Plan

Verwende Papier oder Figma

Kriterien für die Bewertung

  1. Verwendest du Mobile first approach? (1 Pts)
  2. Gibt es eine klare Struktur? (zB.: Header, Footer, Main, Sidebar) (1 Pts)
  3. Is das Design balanced? (1 Pts)
  4. Verwendest du white space einladend / korrekt? (1 Pts)
  5. Ist eine klare Visuelle Führung präsent? (1 Pts)

Hauptseite

Erstelle eine Hauptseite, mit Liste der Kurse. Die Hauptseite muss als Funktionen beinhalten

  1. Liste der Kurse, inkl. Bild, Titel, Kurzbeschreibung
  2. Suchbereich mit freie Text suche (ohne aktive Funktionalität)
  3. Filterfunktion (ohne aktive Funktionalität)
  4. Zusatzinformationen

Egal welche Kurs ich auswähle, lande ich auf die gleichen Details Seite...

Die Teilaufgabe wird bewertet nach:

  1. Korrkete Anwendung von HTML. Öffnende und schließenden Tags sind im richtigen Anzahl, an der richtigen Stellen (1 Pts)
  2. Alle relevanten Inforamtionen sind auf die Webseite dargestellt (1 Pts)
  3. Head Meta-Informationen sind korrekt eingegeben (1 Pts)
  4. Bilder haben einen "alt" Text (1 Pts)
  5. Anwendung von externen CSS File (1 Pts)
  6. CSS Selektoren sind korrekt geschrieben? (3 Pts)
  7. Keine Überflüßige CSS Selektoren sind geblieben? (totes Kode). (1 Pts)
  8. Korrekte Anwendung von Schriftarten (1 Pts)
  9. Korrekte Anwendung von Farben (1 Pts)
  10. Anwendung von CSS Variablen (falls Notwendig) (1 Pts)
  11. Unterstützung von Light und Dark Modes (1 Pts)
  12. Korrekte Anwendung von Flexbox / Grid / Float / Z-index (5 Pts)
  13. Responsive Design: Anwendung von relativen Größen (1 Pts)
  14. Responsive Design: Anwendung von Media Queries (2 Pts)

Bonusaufgabe

Erstelle eine Anmeldeformular für die potenziellen Interessenten. (ohne aktive Funktionalität)
