Aufgabe 0: Installation des Tools und Sample Database

MySQL Community Edition

Um MySQL Community Edition zu installieren, folge diesen Schritten:

  1. Besuche die MySQL Download-Seite.
  2. Wähle die passende Version für dein Betriebssystem aus.
  3. Folge den Installationsanweisungen auf der Webseite.
  4. Lass die Defaultkonfiguration mit dem Port 3306 stehen.
  5. Gebe ein Root Passwort. Das Root Passwort muss auch gemerkt werden

DBeaver Community Edition

DBeaver ist ein universelles Datenbank-Management-Tool, das eine Vielzahl von Datenbanken unterstützt.

Um DBeaver Community Edition zu installieren, folge diesen Schritten:

  1. Besuche die DBeaver Download-Seite.
  2. Wähle die passende Version für dein Betriebssystem aus.
  3. Folge den Installationsanweisungen auf der Webseite.
  4. Starte DBeaver
  5. Du brauchst kein "Sample Database"
  6. Füge eine neue Verbindung zu MYSQL Server ein
  7. Als Treiber wähle MySQL
  8. Gebe das Root Passwort ein
  9. In Eigenschaften musst du AllowPublicKeyRetieval auf TRUE setzen
  10. In Eigenschaften musst du UseSSL auf FALSE setzen
  11. Teste die Verbindung
  12. Wenn alles gut ist, dann klicke auf "Finish"
