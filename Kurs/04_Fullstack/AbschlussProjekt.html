Ziel des Abschlussprojekts

Das Ziel des Abschlussprojekts ist es, euer Wissen und eure Fähigkeiten in einem echten Projekt anzuwenden.

Am Ende des Kurses wird ein Proof of Concept (PoC) entstehen. PoC bedeutet, dass Ideen schrittweise umgesetzt werden, mit Fokus auf die Themen die etwas sichtbares Ergebnis liefern. Ein PoC ist kein fertiges Produkt, sondern ein Konzept, das zeigt, wie die Ideen in der Praxis funktionieren können.

Team

Es hat Sinn verschiedene Rollen anzunehmen. zB: eine Person kümmert sich um das Design, eine Person kümmert sich um die Integration von Änderungen, usw. Die Rollen sind nicht endgültig, sondern können je nach Bedarf angepasst werden.

Aufgabestellung

Erstellt ein komplette Web Applikation, für die Kursverwaltung (KVS). Die wichtigste Ziele sind:

  • Intuitive Navigation
  • Einfache Bearbeitung von Stammdanten (Kurs, Kurstermin, Teilnehmer, Trainer, ...)
  • Semi-Automatische Erstellung von E-Mails an den Teilnehmer / Interessenten ==> Spart wertvolle Arbeitszeit
  • Semi-Automatische Erstellung und Bereitstellung von DOCX-Dokumenten ==> Spart wertvolle Arbeitszeit
  • Kalender und Alert System ==> Wann, was, für welchen Kurs ist zu machen
  • Berechtigungskontrolle durch Gruppen (z.B. Administrator, Programmanager, Read-Only)

Teilaufgaben

Die Teilaufgaben sind parallel auszuführen. Es kann vorkommen, dass Abhängigkeiten entstehen. Aber es ist auch normal.

Entwicklungsumbegung erstellen

  • Erstelle ein Projekt mit next.js und TypeScript. Eventuell aus Auth0 samplen
  • Push it in a private GitHub repository.
  • .gitignore erstellen
  • Projekt in dev Modus starten
  • Einen Branch für eine Entwicklungs-Ziel erstellen.
  • Nach einer Änderung auf dem Entwicklungs-Branch, ein Pull-Request erstellen
  • Änderungen aus dem Entwicklungsbranch mergen

Anmerkung: Gyula wird das Auth0 Konfiguration und GitHub Repository zur Verfügung stellen.

Raumreservierung

Erweitere das System mit Verwaltung von Kursräumen inkl. Raumreservierung

Kurse mit Kurstagen müssen bestimmten Räumen zugewiesen werden. Während der Kurszeiten ist das Raum besetzt.

Zusätzlich ist es möglich Räumlichkeiten für externe Veranstaltungen zu vermieten. Die Externen Partner sind als Stammdaten (Name, Kontaktperson, E-Mail, Telefon, Adresse, Umsatzsteuer-ID) zu verwalten.
