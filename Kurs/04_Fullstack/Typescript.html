Aufgabe 3: Implementieren

Game of Life in TypeScript

Ziel

Erstelle eine webbasierte Implementierung von Conways Game of Life mit TypeScript. Verwende Vite als Build-Tool und Bundler.

Anforderungen

  • Verwende TypeScript für den gesamten Quellcode.
  • Nutze Vite, um dein Projekt zu erstellen und zu bauen.
  • Das Game of Life-Gitter soll im Browser angezeigt und gemäß den Spielregeln aktualisiert werden.
  • Die Gittergröße soll sich an die Bildschirmgröße anpassen.
  • Das Projekt muss responsiv und übersichtlich gestaltet sein.

Schritt-für-Schritt-Anleitung: Neues Vite + TypeScript Projekt aufsetzen

  1. Node.js installieren
    Stelle sicher, dass Node.js installiert ist.
  2. Neues Vite-Projekt erstellen
    Öffne dein Terminal und führe aus:
    npm create vite@latest
    - Gib einen Projektnamen ein (z.B. game-of-life).
    - Wähle Vanilla als Framework.
    - Wähle TypeScript als Variante.
  3. In das Projektverzeichnis wechseln
    cd game-of-life
  4. Abhängigkeiten installieren
    npm install
  5. Entwicklungsserver starten
    npm run dev
    Öffne die angezeigte lokale URL (meist http://localhost:5173/).
  6. "Hello World!" im Browser anzeigen
    Öffne src/main.ts und ersetze den Inhalt durch:
    document.querySelector<HTMLDivElement>('#app')!.innerHTML = `
  <h1>Hello World!</h1>
`;
    Stelle sicher, dass deine index.html ein <div id="app"></div> enthält.
  7. Für Deployment bauen
    npm run build
    Das Ergebnis findest du im dist-Ordner.

Nächste Schritte

  • Ersetze "Hello World!" durch deine Game of Life-Implementierung.
  • Verwende TypeScript für die gesamte Logik und DOM-Manipulation.
  • Mache das Gitter responsiv zur Bildschirmgröße.
