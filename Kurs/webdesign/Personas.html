Aufgabe 1: Recherche

Definition und Begriffe

  • Was ist eine Persona?
  • Woher stammt das Wort?
  • Wie kann ich eine Persona vorstellen?
  • Wie sieht eine Persona aus?

Nutzen

  • Wozu brauchen wir eine Persona?
  • Wozu brauchen wir als Software Entwickler eine Persona?
  • Wie können wir nutzen als Software Entwickler eine Persona?

Prozess der Erstellung

  • Welche Schritte brauche ich, um eine Persona zu erstellen?
  • Welche Aspekte muss ich bei Erstellung einer Persona betrachten?
  • Wie dokumentiere ich meine Persona?

Qualitätsmerkmale

  • Was sind die Merkmale von schwachen/ungeigneten Personas?
  • Was sind die Merkmale von gute Personas?

Aufgabe 2: Eigene Worte finden

Dokumentiere deine Recherege auf deine "Journal" Seite!

  1. Erfahrung in Journal Seite eintragen
  2. mit Git Commit die Änderungen festlegen
  3. mit Git Push die Änderungen auf dem gemeinsame Repository hochladen

Aufgabe 3: Implementieren

Erstelle eine Persona

  1. Erstelle eine erste Version der Persona
  2. Was sind die guten Eigenschaften der Perosna?
  3. Was sind noch weitere Potenziale in der Persona?
  4. Bespreche die Persona, mit einem anderen TN!
  5. (optional) Erweitere/Korrigiere deine Persona!
  6. Stelle die Persona für alle vor.
