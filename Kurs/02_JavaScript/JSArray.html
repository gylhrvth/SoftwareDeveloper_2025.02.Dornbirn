Aufgabe 3: Implementieren

Sample Code

Wähle 7 nützliche JS Array Methoden, und erstelle eine Beispiel-Code in dem du die verwendest.

Einfaches Array

Erstelle eine Funktion mit einem Parameter size erstelle eine neue Array mit der Größe und fülle mit 1, 2, 3, ... bis zu size. Gebe als Ergebnis das Array zurück.

Random Array

Erstelle eine Funktion mit einem Parameter size erstelle eine neue Array mit der Größe und fülle mit zufällige Zahlen zwischen 1 und 100. Gebe als Ergebnis das Array zurück.

Minimum suche

Erstelle eine Funktion minValue(arr), die das kleinste Wert aus dem Array zurückgibt. Löse das Problem mit for schleife und mit forEach

Maximum suche

Erstelle eine Funktion maxValue(arr), die das GRÖßTE Wert aus dem Array zurückgibt. Löse das Problem mit for schleife und mit forEach

Suche nach Position (min)

Erstelle eine Funktion minValueIndex(arr), die der Index des kleinsten Wert aus dem Array zurückgibt. Löse das Problem mit for schleife und mit forEach

Suche nach Position (max)

Erstelle eine Funktion maxValueIndex(arr), die der Index des GRÖßTEN Wert aus dem Array zurückgibt. Löse das Problem mit for schleife und mit forEach

Aufsteigend sortieren

Erstelle eine Funktion sortASC(arr), die Werte aufsteigend sortiert. Verwende die sort() methode.

Absteigend sortieren

Erstelle eine Funktion sortDESC(arr), die Werte absteigend sortiert. Verwende die sort() methode.

Eigene Sort Algorithmus

Erstelle eine Funktion bubbleSort(arr), die Werte aufsteigend sortiert. Verwende die Logik des Bubble Sort.
