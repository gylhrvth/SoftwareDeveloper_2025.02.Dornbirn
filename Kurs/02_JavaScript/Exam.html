Aufgabestellung

Erstelle eine Frontend Applikation für ein Todo App, der den von mir gegebenen API nutzt. Die Kommunikation passier über eine REST API

Hinweis: Nicht alles, waas den API als Information liefert ist relevant für den User.

Das Endergbenis besteht aus:

  1. Eine Single Page Applikation (SPA). Es ist eine Webseite, die mit dem API kommuniziert und direkt in DOM die änderungen durchführt.
  2. Am Anfang sollen alle ToDo's aufgelistet werden
  3. Die Applikation soll regelmäßig die Liste der ToDo's aktualiseren (um neue ToDo's automatisch zu laden)
  4. Fallst eine ToDo ausgewählt wird, soll alle Details dargestellt werden
  5. Neue ToDo's kann zugefügt werden
  6. Bestehenden ToDo's können überarbeitet werden
  7. Bestehenden ToDo's können gelöscht werden.
