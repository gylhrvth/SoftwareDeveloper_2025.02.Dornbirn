C.R.U.D Applikation

Erstelle eine Webseite, die beinhaltet die folgenden Funktionalitäten

  • Create
  • Read
  • Update
  • Delete

Als API musst du dieses restful-api.dev API verwenden.

Der Hintergrund ist eine Webseite, die Lagerobjekte verwalten kann.

Teilaufgabe 1

Besuche die Webseite und lese kurz durch, wie das API auschaut...

Lade in Webbrowser die Liste alle Objekten mit dem URL: https://api.restful-api.dev/objects

Teilaufgabe 2

Erstelle ein neues GUI Element, die Daten aus dem Server holt und für die Benutzer als Liste anzeigt

Falls jmd. auf eine Element klickt, soll die Details geladen und angezeigt werden

Nach dem das Details angezeigt wurden, kann man sinnvolle weise zu der Liste zurück navigieren... In dem Fall wird die Liste aktualisiert.

Teilaufgabe 3

Erstelle ein neues GUI Element, mit dem man eine Element zufügen kann (POST Message).

Falls eine neue Element angelegt wurde, soll die Liste mit alle Objekte neu laden und aktualisiert darstellen

Teilaufgabe 4

Erweitere die Liste so, dass ein Delete Button in der Zeile dazu kommt.

Falls der Benutzer etwas löscht, soll es mit dem DELETE Methode durch API gelöscht werden.

Nach dem Löschen soll die Liste der Objekte aktualisiert werden.

Teilaufgabe 5

Auf dem Detail Seite kann man die Elementen bearbeiten. zB: der Name des Objektes.

Falls etwas geändert hat, gibt es zwei Optionen für der Anwendern:

  1. Speichern - das Objekt wird mit PUT Message auf dem API überschrieben
  2. Abbruch - die Änderungen werden wegworfen, die Detailsseite wird neu geladen

Nach speichern oder abbrechen soll die Liste der Objekte neu geladen und aktualisert werden.
