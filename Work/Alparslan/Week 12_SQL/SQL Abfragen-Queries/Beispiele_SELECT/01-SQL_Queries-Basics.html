SQL Basics - Abfragen

  1. Zeige die aktuelle Uhrzeit als Query-Result an.

    2. SELECT now();

    Lösung: 2025-05-06 09:21:04

  2. Zeige "Hello World!" als Query-Result an.

    3. SELECT 'Hello World' AS message;

    Lösung: Hello World

  3. Zeige alle Spalten, alle Zeilen aus der country Tabelle.

    4. SELECT *
    FROM country;

    Lösung: 238 Länder mit 6 Spalten (Name, Code, Capital, Province, Area, Population)

  4. Zeige alle Spalten, erste 5 Zeilen aus der country Tabelle.

    5. SELECT *
    FROM country
    LIMIT 5;

    Lösung: Die ersten 5 Länder der Tabelle werden angezeigt

  5. Zeige die Spalten Name und Capital der ersten 5 Zeilen aus der country Tabelle.

    6. SELECT Name, Capital
    FROM country
    LIMIT 5;

    Lösung: Gibt die ersten 5 Länder mit deren Hauptstädte aus

  6. Zeige die Spalten Name und Capital der ersten 5 Zeilen aus der country Tabelle mit eigenen Namen.

    7. SELECT Name AS 'Land', Capital AS 'Hauptstadt'
    FROM country
    LIMIT 5;

    Lösung: Gibt die ersten 5 Länder mit deren Hauptstädten aus - mit den Spaltennamen Land und Hauptstadt.

  7. Zeige die Spalten Name und Capital aus der country Tabelle mit Filter auf Name 'Austria'.

    8. SELECT Name, Capital
    FROM country
    WHERE Name = 'Austria';

    Lösung: Gibt den Datensatz für das Land Austria mit der Hauptstadt Vienna zurück.

  8. Zeige die Spalten Name und Capital aus der country Tabelle mit Filter auf Population kleiner 1 Million.

    9. SELECT Name, Capital
    FROM country
    WHERE Population < 1000000;

    Lösung: Es gibt insgesamt 83 Länder mit einer Bevölkerung von weniger als 1 Million. Die Namen der Länder und deren Hauptstädte werden angezeigt.

  9. Zeige die Spalten Name und Capital aus der country Tabelle mit Filter auf Population zwischen 1 Million und 2 Millionen.

    10. SELECT Name, Capital
    FROM country
    WHERE Population BETWEEN 1000000 AND 2000000;

    Lösung: Gibt die Länder mit einer Bevölkerung zwischen 1 Million und 2 Millionen zurück und zeigt deren Hauptstadt an.

  10. Zeige die Spalten Name und Capital aus der country Tabelle mit Filter auf Name beginnend mit 'A'.

    11. SELECT Name, Capital
    FROM country
    WHERE Name LIKE 'A%';

    Lösung: Es gibt insgesamt 14 Länder, deren Name mit dem Buchstaben A beginnt. Die Namen und Hauptstädte dieser Länder werden angezeigt.

  11. Zeige die Spalten Name und Capital aus der country Tabelle mit Filter auf Name am Ende mit 'A'.

    12. SELECT Name, Capital
    FROM country
    WHERE Name LIKE '%A';

    Lösung: Es gibt insgesamt 83 Länder, deren Name mit dem Buchstaben A endet. Die Abfrage gibt deren Namen und Hauptstädte aus.

  12. Zeige die Spalten Name und Capital aus der country Tabelle mit Filter auf Name nicht beginnend mit 'A'.

    13. SELECT Name, Capital
    FROM country
    WHERE Name NOT LIKE 'A%';

    Lösung: Es gibt insgesamt 224 Länder, deren Name nicht mit dem Buchstaben A beginnt. Die Abfrage liefert deren Namen und Hauptstädte.

  13. Zeige die Spalten Name und Capital aus der country Tabelle mit Filter auf Name beginnend mit 'A' oder Population kleiner 1 Million.

    14. SELECT Name, Capital
    FROM country
    WHERE Name LIKE 'A%' OR Population < 1000000

    Lösung: Die Abfrage gibt 92 Länder zurück, deren Name mit dem Buchstaben A beginnt oder deren Bevölkerung unter 1 Million liegt. Angezeigt werden jeweils der Name des Landes und die Hauptstadt.

  14. Zeige die Spalten Name und Capital aus der country Tabelle mit Filter auf Name beginnend mit 'A' und Population kleiner 1 Million.

    15. SELECT Name, Capital
    FROM country
    WHERE Name LIKE 'A%' AND Population < 1000000

    Lösung: Es gibt insgesamt 5 Länder, deren Name mit dem Buchstaben A beginnt und deren Bevölkerung unter 1 Million liegt.
