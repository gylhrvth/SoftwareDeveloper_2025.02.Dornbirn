Web-Entwickler Kurs

Werde ein professioneller Web-Entwickler in nur 5 Monaten!

Details zum Kurs

Kursname: Web-Entwickler: Grundlagen bis Fortgeschrittene Techniken

Beschreibung: In diesem intensiven Kurs lernen Sie alle wichtigen Aspekte der Web-Entwicklung. Von HTML und CSS über JavaScript bis hin zu modernen Frameworks wie React – am Ende dieses Kurses sind Sie bereit, eigene Webseiten und Web-Apps zu entwickeln.

Startdatum: 1. April 2025

Preis: 8. 250 €

Kursdauer: 5 Monate (April bis August 2025)

Woche für Woche - Etappenziele

Woche 1: Einführung in Web-Technologien

  • Grundlagen von HTML
  • Erstellung der ersten HTML-Seite
  • Einführung in CSS für Layout und Design

Woche 2: Fortgeschrittene HTML- und CSS-Techniken

  • Formulare und Eingabefelder
  • Responsive Webdesign mit Media Queries
  • Vertiefung der CSS-Stile und Layouts

Woche 3: Einführung in JavaScript

  • Grundlagen der Programmierung mit JavaScript
  • Variablen, Funktionen und Bedingungen
  • DOM-Manipulation (Ändern von Webseiteninhalten)

Woche 4: Fortgeschrittenes JavaScript

  • Arbeiten mit Arrays und Objekten
  • Einführung in asynchrone Programmierung (Callbacks und Promises)
  • Erstellen interaktiver Webseiten mit JavaScript

Woche 5: Einführung in moderne Web-Frameworks (z. B. React)

  • Erstellung einer einfachen React-Anwendung
  • Verwendung von Komponenten und Props
  • Erste Schritte mit Hooks und Zustandverwaltung

Woche 6-10: Vertiefung von React und Backend-Technologien

  • Erstellen von komplexeren React-Anwendungen
  • Einführung in Node.js und Express
  • Verwendung von APIs und Datenbanken
  • Erstellen einer vollständigen Fullstack-Anwendung

Woche 11-14: Weitere Web-Technologien und Tools

  • Versionierung mit Git und GitHub
  • Einführung in moderne Build-Tools (Webpack, Babel)
  • Erstellen von responsiven und performanten Webseiten

Woche 15-20: Abschlussprojekt und Vorbereitung auf den Berufseinstieg

  • Arbeiten an einem eigenen Abschlussprojekt
  • Vorbereitung auf Vorstellungsgespräche und Portfolio-Erstellung
  • Präsentation des Abschlussprojekts
