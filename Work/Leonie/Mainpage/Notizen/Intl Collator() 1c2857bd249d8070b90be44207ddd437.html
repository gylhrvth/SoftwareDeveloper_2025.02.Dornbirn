Headline

Intl.Collator ist eine JavaScript-API, die für sprach- und regionsspezifische Vergleiche von Strings verwendet wird. Sie ermöglicht es, Strings gemäß den Regeln einer bestimmten Sprache oder Region zu sortieren oder zu vergleichen. Dies ist besonders nützlich, wenn du mit Texten arbeitest, die Sonderzeichen, Akzente oder unterschiedliche Groß- und Kleinschreibung enthalten.

Wie funktioniert Intl.Collator?

Ein Intl.Collator-Objekt wird erstellt, indem du die Sprache und optional zusätzliche Optionen angibst. Dieses Objekt stellt eine Methode namens .compare bereit, die du für Vergleiche oder Sortierungen verwenden kannst.

Syntax

const collator = new Intl.Collator([locales], [options]);
  • locales: Eine Sprach- und Regionsangabe (z. B. "de-DE" für Deutsch in Deutschland oder "en-US" für Englisch in den USA).
  • options: Ein Objekt, das zusätzliche Einstellungen für die Sortierung oder den Vergleich enthält.

Parameter

1. locales

  • Gibt die Sprache und Region an, nach deren Regeln die Strings verglichen werden sollen.
  • Beispiele:
    • "de-DE": Deutsch (Deutschland)
    • "de-AT": Deutsch (Österreich)
    • "en-US": Englisch (USA)

2. options

Ein Objekt mit folgenden möglichen Eigenschaften:

OptionBeschreibung
usage"sort" (Standard) für Sortierung oder "search" für Textsuche.
sensitivityGibt an, wie genau Unterschiede berücksichtigt werden (z. B. Groß-/Kleinschreibung). Werte: "base", "accent", "case", "variant".
ignorePunctuationtrue, um Satzzeichen bei der Sortierung zu ignorieren.
numerictrue, um Zahlen numerisch zu vergleichen (z. B. 2 < 10 statt 2 > 10).
caseFirstGibt an, ob Großbuchstaben zuerst ("upper"), zuletzt ("lower") oder standardmäßig ("false") sortiert werden.
collationGibt die Sortierreihenfolge an (z. B. "phonebk" für Telefonbuchsortierung).

Beispiele

1. Standard-Sortierung

const collator = new Intl.Collator('de-DE');
const words = ['Äpfel', 'Apfel', 'Zebra', 'Ägypten'];
words.sort(collator.compare);
console.log(words);
// Ausgabe: ['Apfel', 'Ägypten', 'Äpfel', 'Zebra']

2. Telefonbuchsortierung

const collator = new Intl.Collator('de-AT', { collation: 'phonebk' });
const words = ['Äpfel', 'Apfel', 'Zebra', 'Ägypten'];
words.sort(collator.compare);
console.log(words);
// Ausgabe: ['Äpfel', 'Ägypten', 'Apfel', 'Zebra']

3. Zahlen numerisch sortieren

const collator = new Intl.Collator('en-US', { numeric: true });
const items = ['item10', 'item2', 'item1'];
items.sort(collator.compare);
console.log(items);
// Ausgabe: ['item1', 'item2', 'item10']

4. Groß-/Kleinschreibung ignorieren

const collator = new Intl.Collator('en-US', { sensitivity: 'base' });
const words = ['apple', 'Apple', 'banana'];
words.sort(collator.compare);
console.log(words);
// Ausgabe: ['apple', 'Apple', 'banana']

Zusammenfassung

  • Intl.Collator ist ideal für sprachspezifische Sortierungen und Vergleiche.
  • Es berücksichtigt regionale Unterschiede, Akzente, Groß-/Kleinschreibung und mehr.
  • Die .compareMethode wird häufig mit Array.sort() verwendet, um Arrays korrekt zu sortieren.
