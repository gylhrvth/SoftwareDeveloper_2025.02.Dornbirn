Was ist Storage in JavaScript?

In JavaScript gibt es zwei Haupttypen von Web Storage: localStorage und sessionStorage. Beide ermöglichen es, Daten im Browser des Benutzers zu speichern, aber sie unterscheiden sich in der Lebensdauer der gespeicherten Daten.

1. localStorage

  • Lebensdauer: Daten bleiben erhalten, auch wenn der Browser geschlossen wird.
  • Speicherort: Daten werden im Browser des Benutzers gespeichert.
  • Kapazität: Typischerweise 5–10 MB pro Domain.
  • Zugriff: Daten sind nur für die gleiche Domain verfügbar.
  • Verwendung: Ideal für Daten, die über mehrere Sitzungen hinweg erhalten bleiben sollen (z. B. Spielstände, Benutzereinstellungen).

Beispiel:

// Speichern eines Werts
localStorage.setItem('key', 'value');

// Abrufen eines Werts
let value = localStorage.getItem('key');
console.log(value); // "value"

// Entfernen eines Werts
localStorage.removeItem('key');

// Löschen aller gespeicherten Daten
localStorage.clear();

2. sessionStorage

  • Lebensdauer: Daten bleiben nur während der aktuellen Browsersitzung erhalten. Sobald der Tab oder das Fenster geschlossen wird, werden die Daten gelöscht.
  • Speicherort: Daten werden im Browser des Benutzers gespeichert.
  • Kapazität: Typischerweise 5–10 MB pro Domain.
  • Zugriff: Daten sind nur für die gleiche Domain und Sitzung verfügbar.
  • Verwendung: Ideal für Daten, die nur während einer Sitzung benötigt werden (z. B. temporäre Formulardaten).

Beispiel:

// Speichern eines Werts
sessionStorage.setItem('key', 'value');

// Abrufen eines Werts
let value = sessionStorage.getItem('key');
console.log(value); // "value"

// Entfernen eines Werts
sessionStorage.removeItem('key');

// Löschen aller gespeicherten Daten
sessionStorage.clear();

Unterschiede zwischen localStorage und sessionStorage

JSON und Storage

Da localStorage und sessionStorage nur Strings speichern können, müssen komplexe Datenstrukturen (wie Objekte oder Arrays) in JSON umgewandelt werden.

Speichern eines Objekts:

let score = { wins: 5, losses: 2, ties: 1 };

// Konvertiere das Objekt in einen JSON-String und speichere es
localStorage.setItem('Score', JSON.stringify(score));

Abrufen eines Objekts:

// Hole den JSON-String und konvertiere ihn zurück in ein Objekt
let score = JSON.parse(localStorage.getItem('Score'));
console.log(score); // { wins: 5, losses: 2, ties: 1 }

Zusammenfassung:

  • localStorage und sessionStorage sind nützliche Tools, um Daten im Browser zu speichern.
  • localStorage speichert Daten dauerhaft, während sessionStorage Daten nur für die aktuelle Sitzung speichert.
  • JSON wird verwendet, um komplexe Datenstrukturen (wie Objekte) in Strings umzuwandeln, da localStorage und sessionStorage nur Strings speichern können.
  • Der Code let score = JSON.parse(localStorage.getItem('Score')); lädt gespeicherte Spielstände aus dem localStorage und konvertiert sie in ein JavaScript-Objekt.

Erklärung von let score = JSON.parse(localStorage.getItem('Score'));

  1. localStorage.getItem('Score'):
    • Diese Methode ruft den Wert ab, der im localStorage unter dem Schlüssel 'Score' gespeichert ist.
    • Der localStorage speichert Daten als Strings. Wenn der Schlüssel 'Score' nicht existiert, gibt getItem() null zurück.
  1. JSON.parse(...):
    • Diese Methode konvertiert einen JSON-String (der aus dem localStorage abgerufen wurde) in ein JavaScript-Objekt.
    • Wenn der Wert unter 'Score' ein JSON-String ist (z. B. {"wins":0,"losses":0,"ties":0}), wird er in ein JavaScript-Objekt umgewandelt:

      { wins: 0, losses: 0, ties: 0 }

  1. let score = ...:
    • Die Variable score speichert das JavaScript-Objekt, das aus dem JSON-String konvertiert wurde.
    • Wenn der Schlüssel 'Score' nicht existiert (d. h., localStorage.getItem('Score') gibt null zurück), wird score auf null gesetzt.

Beispiel:

Angenommen, im localStorage ist folgender Wert unter dem Schlüssel 'Score' gespeichert:

{"wins":5,"losses":2,"ties":1}

Der Code:

let score = JSON.parse(localStorage.getItem('Score'));
console.log(score);

Ergebnis:

{ wins: 5, losses: 2, ties: 1 }

Wenn der Schlüssel 'Score' nicht existiert:

let score = JSON.parse(localStorage.getItem('Score'));
console.log(score);

Ergebnis:

null
