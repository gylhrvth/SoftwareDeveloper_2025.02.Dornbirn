Was ist ein String?

Ein String ist eine Folge von Zeichen, die als Text interpretiert wird. In JavaScript wird ein String durch Anführungszeichen (einfach oder doppelt) definiert.

Was ist der Unterschied zwischen einem nativen Typ und einem Wrapper-Objekt?

Ein nativer Typ (Primitive) ist ein grundlegender Datentyp, der direkt von der Sprache unterstützt wird, wie z.B. string, number, boolean. Ein Wrapper-Objekt ist ein Objekt, das einen nativen Typ umhüllt und zusätzliche Methoden und Eigenschaften bereitstellt, z.B. String, Number, Boolean.

Was macht "text".length?

Die Eigenschaft length gibt die Anzahl der Zeichen in einem String zurück. Für "text" wäre das 4.

Was macht die Funktion concat()? Wie sehen die Parameter aus?

Die Funktion concat() verbindet zwei oder mehr Strings und gibt einen neuen String zurück. Parameter sind die Strings, die verbunden werden sollen.

let result = "Hello".concat(" ", "World"); // "Hello World"

Was macht die Funktion repeat()? Wie sehen die Parameter aus?

Die Funktion repeat() gibt einen neuen String zurück, der mit der angegebenen Anzahl von Kopien des ursprünglichen Strings gefüllt ist. Der Parameter ist die Anzahl der Wiederholungen.

let result = "abc".repeat(3); // "abcabcabc"

Was macht die Funktion charAt()? Wie sehen die Parameter aus?

Die Funktion charAt() gibt das Zeichen an der angegebenen Position im String zurück. Der Parameter ist der Index (nullbasiert).

let result = "hello".charAt(1); // "e"

Was macht die Funktion indexOf()? Wie sehen die Parameter aus?

Die Funktion indexOf() gibt den Index des ersten Vorkommens eines angegebenen Werts im String zurück. Parameter sind der zu suchende Wert und optional der Startindex.

let result = "hello world".indexOf("o"); // 4

Was macht die Funktion lastIndexOf()? Wie sehen die Parameter aus?

Die Funktion lastIndexOf() gibt den Index des letzten Vorkommens eines angegebenen Werts im String zurück. Parameter sind der zu suchende Wert und optional der Startindex.

let result = "hello world".lastIndexOf("o"); // 7

Was macht die Funktion includes()? Wie sehen die Parameter aus?

Die Funktion includes() prüft, ob ein String einen bestimmten Wert enthält, und gibt true oder false zurück. Parameter sind der zu suchende Wert und optional der Startindex.

let result = "hello world".includes("world"); // true

Was macht die Funktion startsWith()? Wie sehen die Parameter aus?

Die Funktion startsWith() prüft, ob ein String mit einem bestimmten Wert beginnt, und gibt true oder false zurück. Parameter sind der zu suchende Wert und optional der Startindex.

let result = "hello world".startsWith("hello"); // true

Was macht die Funktion endsWith()? Wie sehen die Parameter aus?

Die Funktion endsWith() prüft, ob ein String mit einem bestimmten Wert endet, und gibt true oder false zurück. Parameter sind der zu suchende Wert und optional die Länge des Strings.

let result = "hello world".endsWith("world"); // true

Was macht die Funktion substring()? Wie sehen die Parameter aus?

Die Funktion substring() gibt einen Teilstring zwischen zwei Indizes zurück. Parameter sind der Startindex und optional der Endindex.

let result = "hello world".substring(0, 5); // "hello"

Was macht die Funktion toUpperCase()? Wie sehen die Parameter aus?

Die Funktion toUpperCase() gibt einen neuen String zurück, bei dem alle Buchstaben in Großbuchstaben umgewandelt wurden. Sie hat keine Parameter.

let result = "hello".toUpperCase(); // "HELLO"

Was macht die Funktion toLowerCase()? Wie sehen die Parameter aus?

Die Funktion toLowerCase() gibt einen neuen String zurück, bei dem alle Buchstaben in Kleinbuchstaben umgewandelt wurden. Sie hat keine Parameter.

let result = "HELLO".toLowerCase(); // "hello"

Was macht die Funktion trim()? Wie sehen die Parameter aus?

Die Funktion trim() entfernt Leerzeichen am Anfang und Ende eines Strings und gibt einen neuen String zurück. Sie hat keine Parameter.

let result = "  hello  ".trim(); // "hello"

Was macht die Funktion replace()? Wie sehen die Parameter aus?

Die Funktion replace() ersetzt das erste Vorkommen eines angegebenen Werts durch einen neuen Wert und gibt einen neuen String zurück. Parameter sind der zu ersetzende Wert und der neue Wert.

let result = "hello world".replace("world", "there"); // "hello there"

Was macht die Funktion replaceAll()? Wie sehen die Parameter aus?

Die Funktion replaceAll() ersetzt alle Vorkommen eines angegebenen Werts durch einen neuen Wert und gibt einen neuen String zurück. Parameter sind der zu ersetzende Wert und der neue Wert.

let result = "hello world world".replaceAll("world", "there"); // "hello there there"

Was macht die Funktion split()? Wie sehen die Parameter aus?

Die Funktion split() teilt einen String in ein Array von Teilstrings anhand eines angegebenen Trennzeichens. Parameter sind das Trennzeichen und optional die maximale Anzahl der Teilstrings.

let result = "hello world".split(" "); // ["hello", "world"]

Was macht die Funktion filter()? Wie sehen die Parameter aus?

Die Funktion filter() ist eine Methode von Arrays in JavaScript. Sie wird verwendet, um ein neues Array zu erstellen, das nur die Elemente enthält, die eine bestimmte Bedingung erfüllen.

Syntax:

array.filter(callback(element, index, array), thisArg);

Parameter:

  1. callback:
    • Eine Funktion, die für jedes Element des Arrays ausgeführt wird.
    • Diese Funktion muss true zurückgeben, wenn das Element in das neue Array aufgenommen werden soll, und false, wenn es ausgeschlossen werden soll.
    • Der callbackFunktion werden drei Argumente übergeben:
      • element: Das aktuelle Element des Arrays.
      • index (optional): Der Index des aktuellen Elements.
      • array (optional): Das Array, auf dem filter() aufgerufen wurde.
  1. thisArg (optional):
    • Ein Wert, der als this innerhalb der callbackFunktion verwendet wird.

Rückgabewert:

  • Ein neues Array, das nur die Elemente enthält, für die die callbackFunktion true zurückgegeben hat.

Beispiel:

1. Filtern von Zahlen größer als 10:

const numbers = [5, 12, 8, 130, 44];
const filtered = numbers.filter(num => num > 10);
console.log(filtered); // Ausgabe: [12, 130, 44]
  • Die filter()Methode durchläuft jedes Element des Arrays numbers.
  • Nur die Zahlen, die größer als 10 sind, werden in das neue Array filtered aufgenommen.

2. Filtern von Wörtern mit einer bestimmten Bedingung:

const words = ["apple", "banana", "cherry", "date"];
const filteredWords = words.filter(word => word.length > 5);
console.log(filteredWords); // Ausgabe: ["banana", "cherry"]
  • Hier werden nur Wörter mit mehr als 5 Buchstaben in das neue Array filteredWords aufgenommen.
