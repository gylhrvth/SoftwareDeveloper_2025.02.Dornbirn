Pagination

Pagination ist die Technik, große Datenmengen in kleinere, übersichtliche Seiten aufzuteilen. In diesem Projekt wird Pagination verwendet, um die ToDo-Elemente in Seiten aufzuteilen und die Navigation zwischen diesen Seiten zu ermöglichen.

1. Wichtige Variablen

  • currentPage: Die aktuelle Seite, die angezeigt wird. Standardwert ist 1.
    let currentPage = 1;
  • itemsPerPage: Die Anzahl der Elemente, die pro Seite angezeigt werden sollen. Standardwert ist 6.
    const itemsPerPage = 6;

2. Ablauf der Pagination

a) Daten abrufen (getData)

Die Funktion getData ruft die Daten von der API ab und übergibt sie an die Funktion createPagination, die die Pagination-Logik implementiert.

async function getData() {
    try {
        const result = await fetch(ip); // API-Aufruf
        if (result.ok) {
            const data = await result.json();
            createPagination(data); // Pagination erstellen
        } else {
            loadingError(result.status, result.statusText);
        }
    } catch (error) {
        console.error("Network Error:", error);
        alert("Network Error: " + error.message);
    }
}

b) Daten in Seiten aufteilen (createPagination)

Die Funktion createPagination berechnet die Daten, die auf der aktuellen Seite angezeigt werden sollen, und erstellt die Pagination-Steuerelemente.

  1. Berechnung der Gesamtseitenanzahl:
    const totalPages = Math.ceil(data.length / itemsPerPage);
    • data.length: Anzahl der gesamten Daten.
    • itemsPerPage: Anzahl der Elemente pro Seite.
    • Math.ceil: Rundet die Seitenanzahl nach oben.
  1. Berechnung der Daten für die aktuelle Seite:
    const start = (currentPage - 1) * itemsPerPage;
const end = start + itemsPerPage;
const pageData = data.slice(start, end);
    • start: Startindex der Daten für die aktuelle Seite.
    • end: Endindex der Daten für die aktuelle Seite.
    • data.slice(start, end): Extrahiert die Daten für die aktuelle Seite.
  1. DOM und Steuerelemente erstellen:
    • createDOM(pageData): Zeigt die Daten der aktuellen Seite an.
    • createPaginationControls(totalPages): Erstellt die Buttons für die Navigation.

c) Pagination-Steuerelemente erstellen (createPaginationControls)

Diese Funktion erstellt Buttons für die Navigation zwischen den Seiten und zeigt die aktuelle Seite an.

  1. Pagination-Container leeren:
    const paginationContainer = document.querySelector('.pagination');
paginationContainer.innerHTML = '';
  1. "Vorherige Seite"-Button:
    • Deaktiviert, wenn die aktuelle Seite 1 ist.
    • Beim Klick wird currentPage um 1 verringert und die Daten neu geladen.
    const prevButton = document.createElement('button');
prevButton.textContent = 'Vorherige';
prevButton.disabled = currentPage === 1;
prevButton.addEventListener('click', () => {
    currentPage--;
    getData();
});
paginationContainer.appendChild(prevButton);
  1. Seitenanzeige:
    Zeigt die aktuelle Seite und die Gesamtseitenanzahl an.
    const pageInfo = document.createElement('span');
pageInfo.textContent = `Seite ${currentPage} von ${totalPages}`;
paginationContainer.appendChild(pageInfo);
  1. "Nächste Seite"-Button:
    • Deaktiviert, wenn die aktuelle Seite die letzte Seite ist.
    • Beim Klick wird currentPage um 1 erhöht und die Daten neu geladen.
    const nextButton = document.createElement('button');
nextButton.textContent = 'Nächste';
nextButton.disabled = currentPage === totalPages;
nextButton.addEventListener('click', () => {
    currentPage++;
    getData();
});
paginationContainer.appendChild(nextButton);

d) Daten anzeigen (createDOM)

Die Funktion createDOM zeigt die Daten der aktuellen Seite im .items-Container an.

  1. Container leeren:
    const container = document.querySelector('.items');
container.innerHTML = '';
  1. Daten der aktuellen Seite hinzufügen:
    Für jedes Element in
    data wird ein neues DOM-Element erstellt und in den Container eingefügt.
    for (const item of data) {
    const divElement = document.createElement('div');
    divElement.classList.add('item');
    container.appendChild(divElement);

    const name = document.createElement('h2');
    name.textContent = item.title;
    divElement.appendChild(name);
}

3. Beispiel für die Berechnung

Angenommen, es gibt 20 Elemente und itemsPerPage ist 6:

  • Seite 1:
    • start = (1 - 1) * 6 = 0
    • end = 0 + 6 = 6
    • data.slice(0, 6) gibt die Elemente 0 bis 5 zurück.
  • Seite 2:
    • start = (2 - 1) * 6 = 6
    • end = 6 + 6 = 12
    • data.slice(6, 12) gibt die Elemente 6 bis 11 zurück.
  • Seite 3:
    • start = (3 - 1) * 6 = 12
    • end = 12 + 6 = 18
    • data.slice(12, 18) gibt die Elemente 12 bis 17 zurück.
  • Seite 4:
    • start = (4 - 1) * 6 = 18
    • end = 18 + 6 = 24
    • data.slice(18, 24) gibt die Elemente 18 bis 19 zurück (falls nur 20 Elemente vorhanden sind).

4. Zusammenfassung

  • currentPage: Hält die aktuelle Seite.
  • itemsPerPage: Definiert die Anzahl der Elemente pro Seite.
  • createPagination: Teilt die Daten in Seiten auf und zeigt die aktuelle Seite an.
  • createPaginationControls: Erstellt Buttons für die Navigation zwischen den Seiten.
  • createDOM: Zeigt die Daten der aktuellen Seite im DOM an.

Mit dieser Struktur kannst du die Anzahl der Elemente pro Seite einfach anpassen, indem du den Wert von itemsPerPage änderst.
