1. Wofür steht die Abkürzung DOM?

DOM steht für Document Object Model. Es ist eine Programmierschnittstelle, die die Struktur eines HTML- oder XML-Dokuments als Baum darstellt und es ermöglicht, die Inhalte und Struktur des Dokuments zu manipulieren.

2. Was kann ich mit DOM bearbeiten?

  • Inhalte von HTML-Elementen (z. B. Text, HTML-Struktur).
  • Attribute von HTML-Elementen (z. B. id, class, src).
  • CSS-Stile von Elementen.
  • Hinzufügen, Entfernen oder Verschieben von Elementen im DOM-Baum.
  • Ereignisse (z. B. Klicks, Tastendrücke) behandeln.

3. Was kann ich nicht mit DOM bearbeiten?

  • Inhalte außerhalb des Dokuments (z. B. Dateien auf dem Server).
  • Browser-spezifische Einstellungen oder Funktionen direkt ändern.
  • Sicherheitsbeschränkungen umgehen (z. B. Zugriff auf andere Domains).

4. Wie kann ich ein HTML-Element mit einer eindeutigen ID erreichen?

Mit document.getElementById("id").

Beispiel:

const element = document.getElementById("myId");

5. Wie kann ich alle HTML-Elemente erreichen, die eine class="xyz" haben?

Mit document.getElementsByClassName("xyz").

Beispiel:

const elements = document.getElementsByClassName("xyz");

6. Wie kann ich alle HTML-Elemente erreichen, die ich mit einem CSS-Selektor beschreiben kann?

Mit document.querySelectorAll("selector").

Beispiel:

const elements = document.querySelectorAll(".xyz > p");

7. Wie kann ich aus einem Element sein Parent-Element erreichen?

Mit der Eigenschaft parentElement.

Beispiel:

const parent = element.parentElement;

8. Wie kann ich alle Child-Elemente eines Elements erreichen?

Mit der Eigenschaft children.

Beispiel:

const children = element.children;

9. Wie kann ich den Text-Content eines Elements ändern?

Mit der Eigenschaft textContent.

Beispiel:

element.textContent = "Neuer Text";

10. Wie kann ich die Attribute eines Elements ändern?

Mit setAttribute oder direkten Zugriff auf die Eigenschaft.

Beispiel:

element.setAttribute("src", "neuesBild.jpg");
// Oder:
element.src = "neuesBild.jpg";

11. Wie kann ich die Class-Liste eines Elements erreichen?

Mit der Eigenschaft classList.

Beispiel:

const classList = element.classList;

12. Was macht die Funktion classList.add('abc')?

Sie fügt die Klasse abc zum Element hinzu, falls sie noch nicht vorhanden ist.

Beispiel:

element.classList.add("abc");

13. Was macht die Funktion classList.remove('abc')?

Sie entfernt die Klasse abc vom Element, falls sie vorhanden ist.

Beispiel:

element.classList.remove("abc");

14. Was macht die Funktion classList.toggle('abc')?

  • Wenn die Klasse abc vorhanden ist, wird sie entfernt.
  • Wenn die Klasse abc nicht vorhanden ist, wird sie hinzugefügt.
    Beispiel:
element.classList.toggle("abc");

15. Wie erstelle ich ein neues HTML-Element?

Mit document.createElement("tagName").

Beispiel:

const newElement = document.createElement("div");

16. Wie füge ich ein Element an eine gewisse Position des HTML hinzu?

Mit appendChild oder insertBefore.

Beispiel:

parentElement.appendChild(newElement); // Fügt es als letztes Kind hinzu
parentElement.insertBefore(newElement, referenceElement); // Fügt es vor einem Referenzelement ein

17. Wie lösche ich ein HTML-Element?

Mit parentElement.removeChild(childElement) oder element.remove().

Beispiel:

element.remove(); // Direkt löschen
// Oder:
parentElement.removeChild(element);

18. Wie lösche ich ALLE HTML-Elemente aus einem speziellen Element?

Mit der Eigenschaft innerHTML oder einer Schleife.

Beispiel:

element.innerHTML = ""; // Löscht alle Kinder
// Oder:
while (element.firstChild) {
    element.removeChild(element.firstChild);
}
