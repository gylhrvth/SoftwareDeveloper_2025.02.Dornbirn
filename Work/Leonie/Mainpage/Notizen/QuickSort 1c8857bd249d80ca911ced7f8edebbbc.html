Das Prinzip von Quicksort basiert auf dem Teile-und-Herrsche-Ansatz (Divide and Conquer). Es ist ein effizienter, rekursiver Sortieralgorithmus, der ein Array in kleinere Teile aufteilt und diese sortiert. Hier ist eine allgemeine Erklärung:

1. Grundidee

  • Wähle ein Pivot-Element aus dem Array (z. B. das letzte Element, das erste Element oder ein zufälliges Element).
  • Teile das Array in zwei Teile:
    • Linker Teil: Enthält alle Elemente, die kleiner als das Pivot sind.
    • Rechter Teil: Enthält alle Elemente, die größer als das Pivot sind.
  • Das Pivot-Element wird an seine endgültige Position gebracht, da alle kleineren Elemente links und alle größeren rechts davon liegen.
  • Wiederhole diesen Vorgang rekursiv für die beiden Teile (linkes und rechtes Teilarray), bis die Arrays eine Länge von 1 oder 0 haben (diese sind bereits sortiert).

2. Schritte von Quicksort

  1. Pivot-Auswahl:
    • Wähle ein Pivot-Element aus dem Array.
    • Die Wahl des Pivots beeinflusst die Effizienz des Algorithmus (z. B. zufällige Wahl oder Median).
  1. Partitionierung:
    • Teile das Array in zwei Teile:
      • Elemente kleiner als das Pivot kommen in den linken Teil.
      • Elemente größer als das Pivot kommen in den rechten Teil.
    • Das Pivot wird an seine korrekte Position gebracht.
  1. Rekursion:
    • Wende Quicksort rekursiv auf den linken und rechten Teil an.
  1. Zusammenfügen:
    • Die rekursiv sortierten Teile werden zusammengefügt, wobei das Pivot-Element in der Mitte bleibt.

3. Beispiel

Sortiere das Array [8, 3, 1, 7, 0, 10, 2]:

  1. Pivot-Auswahl:
    • Wähle 2 als Pivot (z. B. das letzte Element).
  1. Partitionierung:
    • Linker Teil: [1, 0] (Elemente kleiner als 2).
    • Rechter Teil: [8, 3, 7, 10] (Elemente größer als 2).
    • Pivot 2 wird an seine korrekte Position gebracht: [1, 0, 2, 8, 3, 7, 10].
  1. Rekursion:
    • Sortiere den linken Teil [1, 0]:
      • Pivot: 0.
      • Ergebnis: [0, 1].
    • Sortiere den rechten Teil [8, 3, 7, 10]:
      • Pivot: 10.
      • Ergebnis: [3, 7, 8, 10].
  1. Zusammenfügen:
    • Kombiniere die sortierten Teile: [0, 1] + [2] + [3, 7, 8, 10].
    • Ergebnis: [0, 1, 2, 3, 7, 8, 10].

4. Zeitkomplexität

  • Best Case: \(O(n \log n)\) (wenn das Pivot das Array gleichmäßig teilt).
  • Worst Case: \(O(n^2)\) (wenn das Pivot schlecht gewählt ist, z. B. immer das größte oder kleinste Element).
  • Durchschnitt: \(O(n \log n)\).

5. Vorteile

  • Sehr effizient für große Arrays.
  • In-place-Algorithmus (benötigt keinen zusätzlichen Speicher, wenn die Partitionierung in-place erfolgt).

6. Nachteile

  • Kann im schlechtesten Fall ineffizient sein (\(O(n^2)\)).
  • Die Wahl des Pivot-Elements ist entscheidend für die Leistung.

Quicksort ist einer der am häufigsten verwendeten Sortieralgorithmen, da er in der Praxis oft schneller ist als andere Algorithmen wie Merge Sort oder Bubble Sort.

function quickSort(arr) {
    if (arr.length <= 1) return arr; // Basisfall: Ein Element ist bereits sortiert

    const pivot = arr[arr.length - 1]; // Wähle das letzte Element als Pivot
    const left = arr.filter(el => el < pivot); // Elemente kleiner als Pivot
    const right = arr.filter(el => el > pivot); // Elemente größer als Pivot

    return [...quickSort(left), pivot, ...quickSort(right)]; // Rekursiv sortieren und zusammenfügen
}

// Beispiel
const array = [8, 3, 1, 7, 0, 10, 2];
const sortedArray = quickSort(array);
console.log("Sorted Array:", sortedArray);

1. Funktionsweise von quickSort

a) Basisfall

if (arr.length <= 1) return arr;
  • Wenn das Array nur ein Element oder leer ist (arr.length <= 1), ist es bereits sortiert.
  • Die Funktion gibt das Array direkt zurück.

b) Pivot-Auswahl

const pivot = arr[arr.length - 1];
  • Das letzte Element des Arrays wird als Pivot ausgewählt.
  • Das Pivot dient als Referenzpunkt, um das Array in zwei Teile zu teilen:
    • Linker Teil: Alle Elemente kleiner als das Pivot.
    • Rechter Teil: Alle Elemente größer als das Pivot.

c) Partitionierung

const left = arr.filter(el => el < pivot);
const right = arr.filter(el => el > pivot);
  • arr.filter(el => el < pivot):
    • Erstellt ein neues Array left, das alle Elemente enthält, die kleiner als das Pivot sind.
  • arr.filter(el => el > pivot):
    • Erstellt ein neues Array right, das alle Elemente enthält, die größer als das Pivot sind.

d) Rekursion

return [...quickSort(left), pivot, ...quickSort(right)];
  • Die Funktion ruft sich selbst rekursiv auf:
    • quickSort(left): Sortiert den linken Teil des Arrays.
    • quickSort(right): Sortiert den rechten Teil des Arrays.
  • Die Ergebnisse werden zusammengefügt:
    • [...quickSort(left), pivot, ...quickSort(right)]:
      • Das sortierte linke Array.
      • Das Pivot-Element.
      • Das sortierte rechte Array.
