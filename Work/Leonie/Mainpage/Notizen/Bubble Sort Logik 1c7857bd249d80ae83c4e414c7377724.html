Der Bubble Sort ist ein einfacher Sortieralgorithmus, der durch wiederholtes Vergleichen und Vertauschen benachbarter Elemente ein Array sortiert. Die Logik basiert darauf, dass größere Werte "nach oben wandern" (wie Luftblasen im Wasser), bis das Array vollständig sortiert ist.

Funktionsweise von Bubble Sort:

  1. Vergleiche benachbarte Elemente:
    • Gehe durch das Array und vergleiche jeweils zwei benachbarte Elemente.
    • Wenn das linke Element größer ist als das rechte, vertausche sie.
  1. Wiederhole den Prozess:
    • Wiederhole den Vergleich und das Vertauschen für alle Elemente im Array.
    • Nach jedem Durchlauf ist das größte Element an seiner endgültigen Position (am Ende des Arrays).
  1. Reduziere den Bereich:
    • Da das größte Element nach jedem Durchlauf an der richtigen Position ist, wird der Bereich, der sortiert werden muss, kleiner.
  1. Beende den Algorithmus:
    • Der Algorithmus endet, wenn keine Vertauschungen mehr nötig sind (das Array ist sortiert).

Schritt-für-Schritt-Beispiel:

Angenommen, wir haben das Array: [5, 3, 8, 1, 2].

Durchlauf 1:

  • Vergleiche 5 und 3: Tausche → [3, 5, 8, 1, 2]
  • Vergleiche 5 und 8: Kein Tausch → [3, 5, 8, 1, 2]
  • Vergleiche 8 und 1: Tausche → [3, 5, 1, 8, 2]
  • Vergleiche 8 und 2: Tausche → [3, 5, 1, 2, 8]

Das größte Element 8 ist jetzt an der richtigen Position.

Durchlauf 2:

  • Vergleiche 3 und 5: Kein Tausch → [3, 5, 1, 2, 8]
  • Vergleiche 5 und 1: Tausche → [3, 1, 5, 2, 8]
  • Vergleiche 5 und 2: Tausche → [3, 1, 2, 5, 8]

Das zweitgrößte Element 5 ist jetzt an der richtigen Position.

Durchlauf 3:

  • Vergleiche 3 und 1: Tausche → [1, 3, 2, 5, 8]
  • Vergleiche 3 und 2: Tausche → [1, 2, 3, 5, 8]

Das drittgrößte Element 3 ist jetzt an der richtigen Position.

Durchlauf 4:

  • Vergleiche 1 und 2: Kein Tausch → [1, 2, 3, 5, 8]

Das Array ist jetzt vollständig sortiert.

Code-Logik:

function bubbleSort(arr) {
    const n = arr.length;
    for (let i = 0; i < n - 1; i++) { // Äußere Schleife: Anzahl der Durchläufe
        for (let j = 0; j < n - i - 1; j++) { // Innere Schleife: Vergleiche benachbarte Elemente
            if (arr[j] > arr[j + 1]) { // Wenn das linke Element größer ist, tausche
                [arr[j], arr[j + 1]] = [arr[j + 1], arr[j]]; // Swap
            }
        }
    }
    return arr; // Rückgabe des sortierten Arrays
}

Komplexität:

  • Zeitkomplexität:
    • Best Case (bereits sortiert): O(n) (mit Optimierung, wenn keine Vertauschungen mehr nötig sind).
    • Schlechtester Fall: O(n²) (bei vollständig unsortiertem Array).
  • Raumkomplexität: O(1) (in-place Sortierung, keine zusätzlichen Datenstrukturen).

Zusammenfassung:

  • Bubble Sort ist einfach zu implementieren, aber ineffizient für große Arrays.
  • Es eignet sich gut für kleine Datensätze oder als Lehrbeispiel für Sortieralgorithmen.
  • Größere Sortieralgorithmen wie Merge Sort oder Quick Sort sind für größere Arrays deutlich effizienter.

Funktionsweise:

  1. Initialisierung:
    • Die Länge des Arrays wird in n gespeichert.
    • Es gibt zwei verschachtelte Schleifen:
      • Die äußere Schleife (for (let i = 0; i < n - 1; i++)) steuert die Anzahl der Durchläufe.
      • Die innere Schleife (for (let j = 0; j < n - i - 1; j++)) führt die Vergleiche und Vertauschungen durch.
  1. Vergleich und Vertauschung:
    • Die innere Schleife vergleicht jedes Paar benachbarter Elemente (arr[j] und arr[j + 1]).
    • Wenn das linke Element (arr[j]) größer ist als das rechte (arr[j + 1]), werden die beiden Elemente vertauscht.
    • Der Tausch erfolgt mit der Destrukturierungssyntax:

      [arr[j], arr[j + 1]] = [arr[j + 1], arr[j]];

  1. Reduzierung des Bereichs:
    • Nach jedem Durchlauf der äußeren Schleife ist das größte Element an seiner endgültigen Position (am Ende des Arrays).
    • Daher wird der Bereich, der sortiert werden muss, mit jedem Durchlauf kleiner (n - i - 1).
  1. Rückgabe:
    • Nach Abschluss aller Durchläufe gibt die Funktion das sortierte Array zurück.

Beispiel:

Angenommen, das Array ist arr = [5, 3, 8, 1, 2].

Durchlauf 1 (i = 0):

  • Vergleiche 5 und 8: Kein Tausch → [3, 5, 8, 1, 2]
