Headline

1. Was ist ein JavaScript Date Object?

Das Date-Objekt in JavaScript ist ein integriertes Objekt, das verwendet wird, um Datum und Uhrzeit zu speichern und zu manipulieren. Es bietet Methoden, um das aktuelle Datum und die Uhrzeit abzurufen, zu formatieren und zu berechnen.

2. Welche Möglichkeiten gibt es, ein Datum anzuzeigen?

Ein Datum kann in JavaScript auf verschiedene Arten angezeigt werden, z. B.:

  • toString(): Gibt das Datum als lesbare Zeichenkette zurück.
  • toDateString(): Gibt nur das Datum (ohne Zeit) als lesbare Zeichenkette zurück.
  • toISOString(): Gibt das Datum im ISO 8601-Format zurück (z. B. 2025-04-03T12:00:00.000Z).
  • toLocaleDateString(): Gibt das Datum in einem lokalisierten Format zurück.
  • toLocaleString(): Gibt Datum und Uhrzeit in einem lokalisierten Format zurück.
  • getFullYear(), getMonth(), getDate(): Einzelne Teile des Datums können abgerufen und kombiniert werden.

3. Was ist die "Zero Time" bei der JS Date API?

Die "Zero Time" ist der Zeitpunkt 1. Januar 1970, 00:00:00 UTC. Dieser Zeitpunkt wird als Epoche oder Unix-Zeit bezeichnet. Alle Zeitangaben in JavaScript werden relativ zu diesem Zeitpunkt in Millisekunden gemessen.

4. Mit welchem Input kann man ein new DateObjekt erstellen?

Ein Date-Objekt kann mit verschiedenen Eingaben erstellt werden:

  • Ohne Argumente: new Date() erstellt ein Objekt mit dem aktuellen Datum und der aktuellen Uhrzeit.
  • Zeitstempel (Millisekunden seit 1970): new Date(0) erstellt ein Datum basierend auf einem Zeitstempel.
  • Datum als String: new Date("2025-04-03") erstellt ein Datum aus einem ISO 8601-String.
  • Einzelne Werte: new Date(year, month, day, hours, minutes, seconds, milliseconds) erstellt ein Datum aus einzelnen Werten (Monat beginnt bei 0).

5. Wie werden in der JS Date API die Monate gezählt?

Die Monate werden nullbasiert gezählt:

  • Januar = 0
  • Februar = 1
  • ...
  • Dezember = 11

6. Was passiert bei der "falschen" Eingabe von Tag oder Monat?

Wenn ein ungültiger Tag oder Monat eingegeben wird:

  • Automatische Korrektur: JavaScript passt das Datum automatisch an. Beispiel:
    new Date(2025, 13, 1); // Ergebnis: 1. Januar 2026
new Date(2025, 1, 32); // Ergebnis: 4. März 2025
  • Ungültiges Datum: Wenn die Eingabe nicht interpretiert werden kann, gibt new Date() ein Invalid DateObjekt zurück. Beispiel:
    new Date("invalid-date"); // Ergebnis: Invalid Date

    Erstellen eines Date-Objekts

    • new Date()
      Erstellt ein neues
      DateObjekt mit dem aktuellen Datum und der aktuellen Uhrzeit.
      const now = new Date();
console.log(now); // Beispiel: Thu Apr 03 2025 14:30:00 GMT+0200

    Getter-Methoden (Abrufen von Datum und Uhrzeit)

    • getDate()

      Gibt den Tag des Monats (1–31) zurück.

      const date = new Date();
console.log(date.getDate()); // Beispiel: 3
    • getDay()

      Gibt den Wochentag als Zahl zurück (0 = Sonntag, 1 = Montag, ..., 6 = Samstag).

      const date = new Date();
console.log(date.getDay()); // Beispiel: 4 (Donnerstag)
    • getFullYear()

      Gibt das Jahr als vierstellige Zahl zurück.

      const date = new Date();
console.log(date.getFullYear()); // Beispiel: 2025
    • getMonth()

      Gibt den Monat zurück (0 = Januar, 1 = Februar, ..., 11 = Dezember).

      const date = new Date();
console.log(date.getMonth()); // Beispiel: 3 (April)
    • getHours()

      Gibt die Stunde (0–23) zurück.

      const date = new Date();
console.log(date.getHours()); // Beispiel: 14
    • getMinutes()

      Gibt die Minuten (0–59) zurück.

      const date = new Date();
console.log(date.getMinutes()); // Beispiel: 30
    • getSeconds()

      Gibt die Sekunden (0–59) zurück.

      const date = new Date();
console.log(date.getSeconds()); // Beispiel: 45
    • getMilliseconds()

      Gibt die Millisekunden (0–999) zurück.

      const date = new Date();
console.log(date.getMilliseconds()); // Beispiel: 123
    • getTime()

      Gibt die Anzahl der Millisekunden seit dem 1. Januar 1970, 00:00:00 UTC zurück.

      const date = new Date();
console.log(date.getTime()); // Beispiel: 1740984600000

    Beispiel Geburtstag ausrechnen:

    Vordefiniert Variarblen:

    let date = "";
let dateInput = "";
let day = "";
let month = "";
let year = "";
let inputDateValue = "";

    1. const today = new Date();

    • Erstellt ein neues DateObjekt, das das aktuelle Datum und die aktuelle Uhrzeit repräsentiert.
    • Beispiel: Wenn heute der 3. April 2025 ist, enthält today den Wert Thu Apr 03 2025 00:00:00 GMT+0000.

    2. const birthday = new Date(today.getFullYear(), dateInput.getMonth(), dateInput.getDate());

    • Erstellt ein neues DateObjekt für den Geburtstag im aktuellen Jahr.
    • today.getFullYear(): Holt das aktuelle Jahr (z. B. 2025).
    • dateInput.getMonth(): Holt den Monat des Geburtstags aus dateInput (0-basiert, z. B. 6 für Juli).
    • dateInput.getDate(): Holt den Tag des Geburtstags aus dateInput (z. B. 29).
    • Beispiel: Wenn der Geburtstag der 29. Juli ist, wird birthday auf Tue Jul 29 2025 00:00:00 GMT+0000 gesetzt.

    3. if (today > birthday) { birthday.setFullYear(today.getFullYear() + 1); }

    • Überprüft, ob der Geburtstag in diesem Jahr bereits vergangen ist:
      • today > birthday: Wenn das heutige Datum nach dem Geburtstag liegt, ist der Geburtstag in diesem Jahr vorbei.
    • Wenn der Geburtstag vorbei ist, wird das Jahr des Geburtstags um 1 erhöht:
      • birthday.setFullYear(today.getFullYear() + 1): Setzt den Geburtstag auf das nächste Jahr.
    • Beispiel:
      • Wenn heute der 3. August 2025 ist und der Geburtstag der 29. Juli 2025 war, wird birthday auf Tue Jul 29 2026 00:00:00 GMT+0000 gesetzt.

    4. const timeDiff = birthday - today;

    • Berechnet die Zeitdifferenz zwischen dem heutigen Datum (today) und dem nächsten Geburtstag (birthday) in Millisekunden.
    • Beispiel:
      • Wenn heute der 3. April 2025 ist und der nächste Geburtstag der 29. Juli 2025, beträgt die Differenz etwa 10368000000 Millisekunden.

    5. const daysToBirthday = Math.ceil(timeDiff / (1000 * 60 * 60 * 24));

    • Wandelt die Zeitdifferenz von Millisekunden in Tage um:
      • 1000: Millisekunden in Sekunden.
      • 60: Sekunden in Minuten.
      • 60: Minuten in Stunden.
      • 24: Stunden in Tage.
    • Math.ceil(): Rundet die Anzahl der Tage auf die nächste ganze Zahl auf.
    • Beispiel:
      • Wenn die Zeitdifferenz 10368000000 Millisekunden beträgt, ergibt die Berechnung 120 Tage.

    6. console.log(daysToBirthday);

    • Gibt die berechnete Anzahl der Tage bis zum nächsten Geburtstag in der Konsole aus.
    • Beispiel: 120.

    7. const daysOutput = document.getElementById("feedback");

    • Holt das HTML-Element mit der ID feedback, in dem die Ausgabe angezeigt werden soll.
    • Beispiel: <div id="feedback"></div>.

    8. const daysParagraph = document.createElement("p");

    • Erstellt ein neues <p>Element (Absatz), das den Text mit der Anzahl der Tage enthalten wird.

    9. daysParagraph.textContent = ...

    • Setzt den Textinhalt des neuen <p>Elements.
    • Beispiel: daysParagraph.textContent = "Es sind noch 120 Tage bis zu deinem Geburtstag.";.

    10. daysOutput.innerHTML = "";

    • Löscht den vorherigen Inhalt des Elements mit der ID feedback, um sicherzustellen, dass nur die aktuelle Berechnung angezeigt wird.

    11. daysOutput.appendChild(daysParagraph);

    • Fügt das neue <p>Element (daysParagraph) in das feedbackElement ein.
    • Beispiel:
      • Vorher: <div id="feedback"></div>
      • Nachher: <div id="feedback"><p>Es sind noch 120 Tage bis zu deinem Geburtstag.</p></div>

    Zusammenfassung:

    Dieser Code berechnet die Anzahl der Tage bis zum nächsten Geburtstag und zeigt das Ergebnis in einem HTML-Element (feedback) an. Wenn der Geburtstag in diesem Jahr bereits vergangen ist, wird der nächste Geburtstag im nächsten Jahr berücksichtigt.

Setter-Methoden (Ändern von Datum und Uhrzeit)

  • setDate(day)

    Setzt den Tag des Monats (1–31).

    const date = new Date();
date.setDate(15);
console.log(date); // Beispiel: Tue Apr 15 2025
  • setFullYear(year)

    Setzt das Jahr.

    const date = new Date();
date.setFullYear(2030);
console.log(date); // Beispiel: Thu Apr 03 2030
  • setMonth(month)

    Setzt den Monat (0 = Januar, 1 = Februar, ..., 11 = Dezember).

    const date = new Date();
date.setMonth(5); // Juni
console.log(date); // Beispiel: Mon Jun 03 2025
  • setHours(hours)

    Setzt die Stunde (0–23).

    const date = new Date();
date.setHours(10);
console.log(date); // Beispiel: Thu Apr 03 2025 10:30:00
  • setMinutes(minutes)

    Setzt die Minuten (0–59).

    const date = new Date();
date.setMinutes(45);
console.log(date); // Beispiel: Thu Apr 03 2025 14:45:00
  • setSeconds(seconds)

    Setzt die Sekunden (0–59).

    const date = new Date();
date.setSeconds(20);
console.log(date); // Beispiel: Thu Apr 03 2025 14:30:20
  • setMilliseconds(milliseconds)

    Setzt die Millisekunden (0–999).

    const date = new Date();
date.setMilliseconds(500);
console.log(date); // Beispiel: Thu Apr 03 2025 14:30:00.500
  • setTime(milliseconds)

    Setzt das Datum basierend auf der Anzahl der Millisekunden seit dem 1. Januar 1970, 00:00:00 UTC.

    const date = new Date();
date.setTime(1740984600000);
console.log(date); // Beispiel: Thu Apr 03 2025 14:30:00

Formatierungs-Methoden

  • toString()

    Gibt das Datum als lesbare Zeichenkette zurück.

    const date = new Date();
console.log(date.toString()); // Beispiel: Thu Apr 03 2025 14:30:00 GMT+0200
  • toDateString()

    Gibt nur das Datum (ohne Zeit) als lesbare Zeichenkette zurück.

    const date = new Date();
console.log(date.toDateString()); // Beispiel: Thu Apr 03 2025
  • toUTCString()

    Gibt das Datum und die Uhrzeit in UTC als Zeichenkette zurück.

    const date = new Date();
console.log(date.toUTCString()); // Beispiel: Thu, 03 Apr 2025 12:30:00 GMT
  • toISOString()

    Gibt das Datum im ISO 8601-Format zurück.

    const date = new Date();
console.log(date.toISOString()); // Beispiel: 2025-04-03T12:30:00.000Z
