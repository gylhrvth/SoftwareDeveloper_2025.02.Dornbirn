EJS (Embedded JavaScript) ist eine flexible und einfache Template-Engine, die es ermöglicht, dynamische Inhalte in HTML-Seiten zu rendern. Hier sind einige Dinge, die du mit EJS machen kannst:

1. Variablen einfügen

Du kannst Variablen aus deinem Server in die HTML-Seite einfügen:

<h1>Willkommen, <%= username %>!</h1>

Beispiel im Server:

app.get('/', (req, res) => {
  res.render('index', { username: 'Leo' });
});

2. Schleifen

EJS unterstützt Schleifen, um Listen oder Arrays zu rendern:

<ul>
  <% items.forEach(item => { %>
    <li><%= item %></li>
  <% }); %>
</ul>

Beispiel im Server:

app.get('/', (req, res) => {
  res.render('index', { items: ['Apfel', 'Banane', 'Kirsche'] });
});

3. Bedingte Anweisungen

Du kannst Bedingungen verwenden, um Inhalte dynamisch anzuzeigen:

<% if (isLoggedIn) { %>
  <p>Willkommen zurück!</p>
<% } else { %>
  <p>Bitte melde dich an.</p>
<% } %>

Beispiel im Server:

app.get('/', (req, res) => {
  res.render('index', { isLoggedIn: true });
});

4. Partials (Wiederverwendbare Komponenten)

Du kannst wiederverwendbare Teile wie Header, Footer oder Navigation erstellen und in mehreren Seiten einfügen:

<%- include('partials/header') %>
<h1>Inhalt der Seite</h1>
<%- include('partials/footer') %>

Beispiel:

  • views/partials/header.ejs:
    <header>
  <h1>Meine Website</h1>
</header>
  • views/partials/footer.ejs:
    <footer>
  <p>&copy; 2025 Meine Website</p>
</footer>

5. Formulare und Benutzereingaben

Du kannst Formulare erstellen und Benutzereingaben an den Server senden:

<form action="/submit" method="POST">
  <input type="text" name="username" placeholder="Benutzername">
  <button type="submit">Absenden</button>
</form>

Beispiel im Server:

app.post('/submit', (req, res) => {
  const username = req.body.username;
  res.render('welcome', { username });
});

6. Dynamische CSS-Klassen

Du kannst CSS-Klassen dynamisch basierend auf Variablen setzen:

<div class="<%= isActive ? 'active' : 'inactive' %>">
  Inhalt
</div>

7. Daten aus einer Datenbank anzeigen

EJS kann Daten aus einer Datenbank rendern:

<table>
  <% users.forEach(user => { %>
    <tr>
      <td><%= user.name %></td>
      <td><%= user.email %></td>
    </tr>
  <% }); %>
</table>

Beispiel im Server:

app.get('/users', async (req, res) => {
  const users = await db.query('SELECT name, email FROM users');
  res.render('users', { users });
});

8. Fehlerseiten

Du kannst benutzerdefinierte Fehlerseiten erstellen:

<h1><%= title %></h1>
<p>Die Seite wurde nicht gefunden.</p>

Beispiel im Server:

app.use((req, res) => {
  res.status(404).render('404', { title: 'Seite nicht gefunden' });
});

9. Lokale und globale Variablen

  • Lokale Variablen: Werden nur für eine bestimmte Route übergeben.
  • Globale Variablen: Können mit Middleware definiert werden und stehen in allen Templates zur Verfügung:
app.use((req, res, next) => {
  res.locals.siteName = 'Meine Website';
  next();
});

In EJS:

<h1>Willkommen auf <%= siteName %></h1>

10. Layouts mit EJS

EJS unterstützt keine Layouts direkt, aber du kannst sie mit Partials simulieren:

<%- include('partials/header') %>
<main>
  <%- body %>
</main>
<%- include('partials/footer') %>

Beispiel im Server:

app.get('/', (req, res) => {
  res.render('layout', { body: '<h1>Startseite</h1>' });
});

Zusammenfassung

Mit EJS kannst du:

  • Dynamische Inhalte einfügen.
  • Schleifen und Bedingungen verwenden.
  • Wiederverwendbare Komponenten (Partials) erstellen.
  • Daten aus einer Datenbank anzeigen.
  • Formulare und Benutzereingaben verarbeiten.
  • Fehlerseiten und Layouts erstellen.

EJS ist einfach und flexibel, ideal für kleine bis mittelgroße Projekte.
