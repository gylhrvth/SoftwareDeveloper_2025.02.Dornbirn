JSON (JavaScript Object Notation)

JSON ist ein leichtgewichtiges Datenformat, das häufig für den Datenaustausch zwischen einem Server und einer Webanwendung verwendet wird. Es basiert auf einer einfachen, textbasierten Struktur, die leicht von Menschen lesbar und von Maschinen analysierbar ist.

Wichtige Eigenschaften von JSON

  1. Textbasiert:
    • JSON ist ein reines Textformat, das leicht in Dateien gespeichert oder über Netzwerke übertragen werden kann.
  1. Struktur:
    • JSON-Daten bestehen aus Schlüssel-Wert-Paaren (ähnlich wie Objekte in JavaScript).
    • Es unterstützt verschachtelte Strukturen wie Arrays und Objekte.
  1. Kompatibilität:
    • JSON ist unabhängig von einer Programmiersprache und wird von fast allen modernen Programmiersprachen unterstützt.
  1. Verwendung:
    • JSON wird häufig in APIs (z. B. RESTful APIs) verwendet, um Daten zwischen Client und Server auszutauschen.

JSON-Syntax

  1. Objekte:
    • JSON-Objekte bestehen aus Schlüssel-Wert-Paaren.
    • Schlüssel sind immer Strings (in doppelten Anführungszeichen).
    • Werte können Strings, Zahlen, Arrays, Objekte, true, false oder null sein.

    Beispiel:

    {
    "name": "John",
    "age": 30,
    "isStudent": false,
    "skills": ["JavaScript", "HTML", "CSS"],
    "address": {
        "city": "Berlin",
        "zip": "10115"
    }
}
  1. Arrays:
    • JSON-Arrays sind geordnete Listen von Werten.
    • Die Werte können verschiedene Datentypen haben.

    Beispiel:

    [
    "Apple",
    "Banana",
    "Cherry"
]

JSON in JavaScript

1. JSON-String in ein JavaScript-Objekt umwandeln

Verwende JSON.parse(), um einen JSON-String in ein JavaScript-Objekt zu konvertieren.

Beispiel:

const jsonString = '{"name": "John", "age": 30}';
const obj = JSON.parse(jsonString);

console.log(obj.name); // Ausgabe: John
console.log(obj.age);  // Ausgabe: 30

2. JavaScript-Objekt in einen JSON-String umwandeln

Verwende JSON.stringify(), um ein JavaScript-Objekt in einen JSON-String zu konvertieren.

Beispiel:

const obj = { name: "John", age: 30 };
const jsonString = JSON.stringify(obj);

console.log(jsonString); // Ausgabe: {"name":"John","age":30}

Wichtige Punkte zu beachten

  1. Doppelte Anführungszeichen:
    • JSON-Schlüssel und -Strings müssen in doppelten Anführungszeichen (") stehen. Einzelne Anführungszeichen (') sind nicht erlaubt.

    Richtig:

    { "name": "John" }

    Falsch:

    { 'name': 'John' }
  1. Keine Kommentare:
    • JSON erlaubt keine Kommentare. Wenn du Kommentare benötigst, musst du sie außerhalb des JSON-Dokuments hinzufügen.
  1. Datenformat:
    • JSON unterstützt nur bestimmte Datentypen:
      • Strings
      • Zahlen
      • Objekte
      • Arrays
      • true, false, null

    Nicht unterstützt:

    • Funktionen, undefined, oder andere komplexe Datentypen.
  1. Sicherheitsaspekte:
    • JSON-Daten sollten immer aus vertrauenswürdigen Quellen stammen. Unsicheres Parsen von JSON kann zu Sicherheitslücken führen (z. B. Cross-Site Scripting).

Praktische Beispiele

JSON von einer API abrufen

fetch('<https://api.example.com/data>')
    .then(response => response.json()) // JSON-String in ein Objekt umwandeln
    .then(data => {
        console.log(data); // Verarbeite die JSON-Daten
    })
    .catch(error => console.error('Fehler:', error));

JSON in einer Datei speichern

JSON wird häufig in .json-Dateien gespeichert. Beispiel für eine Datei data.json:

{
    "users": [
        { "id": 1, "name": "Alice" },
        { "id": 2, "name": "Bob" }
    ]
}

In JavaScript kannst du diese Datei mit einem HTTP-Request oder einem Node.js-Modul laden.

Zusammenfassung

  • JSON ist ein einfaches, textbasiertes Datenformat für den Datenaustausch.
  • Verwende JSON.parse() und JSON.stringify() in JavaScript, um zwischen JSON und JavaScript-Objekten zu konvertieren.
  • Achte auf die JSON-Syntax (z. B. doppelte Anführungszeichen, keine Kommentare).
  • JSON ist ideal für APIs und wird in vielen modernen Webanwendungen verwendet.
