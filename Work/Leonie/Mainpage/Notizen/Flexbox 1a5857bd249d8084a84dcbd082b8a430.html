Flexbox (Flexible Box Layout) ist ein CSS-Layout-Modul, das eine effizientere Möglichkeit bietet, Layouts zu erstellen und zu gestalten. Es ermöglicht das Anordnen von Elementen in einem Container, selbst wenn deren Größe unbekannt oder dynamisch ist. Hier sind die grundlegenden Konzepte und Eigenschaften von Flexbox:

Grundlegende Konzepte

  1. Flex Container: Das übergeordnete Element, das die Flexbox-Eigenschaften enthält. Es wird durch das Setzen der CSS-Eigenschaft display auf flex oder inline-flex definiert.
  1. Flex Items: Die direkten Kinder des Flex Containers, die innerhalb des Containers angeordnet werden.

Wichtige Eigenschaften des Flex Containers

  • display: flex; oder display: inline-flex;: Definiert ein Element als Flex Container.
  • flex-direction: Bestimmt die Hauptachse (row, row-reverse, column, column-reverse).
  • justify-content: Richtet die Flex Items entlang der Hauptachse aus (flex-start, flex-end, center, space-between, space-around, space-evenly).
  • align-items: Richtet die Flex Items entlang der Querachse aus (flex-start, flex-end, center, baseline, stretch).
  • flex-wrap: Bestimmt, ob die Flex Items in eine neue Zeile umbrechen (nowrap, wrap, wrap-reverse).

Wichtige Eigenschaften der Flex Items

  • order: Bestimmt die Reihenfolge der Flex Items.
  • flex-grow: Bestimmt, wie viel ein Flex Item wachsen kann, relativ zu den anderen Flex Items.
  • flex-shrink: Bestimmt, wie viel ein Flex Item schrumpfen kann, relativ zu den anderen Flex Items.
  • flex-basis: Bestimmt die anfängliche Größe eines Flex Items.
  • align-self: Überschreibt die align-itemsEigenschaft für ein einzelnes Flex Item.

In diesem Beispiel wird ein Flex Container erstellt, der fünf Flex Items enthält. Die Flex Items werden in einer Reihe (flex-direction: row) angeordnet und gleichmäßig verteilt (justify-content: space-around). Die Flex Items sind zentriert (align-items: center) und haben eine feste Höhe (height: 100vh).


Unterschied zwischen Flexbox und Grid

  • Flexbox: Ein-dimensionales Layout-Modul, ideal für Layouts in einer einzigen Dimension (Zeile oder Spalte). Bietet leistungsstarke Ausrichtungs- und Verteilungsfunktionen.
  • Grid: Zwei-dimensionales Layout-Modul, ideal für komplexe Layouts, die sowohl Zeilen als auch Spalten umfassen. Ermöglicht die explizite Platzierung von Elementen in einem Raster.

Beide Module können auch zusammen verwendet werden, um noch flexiblere und komplexere Layouts zu erstellen.

Erklärung zu einem Beispiel:

.content .box {
    flex: 1 1 calc(50% - var(--padding));
    background-color: lightgray;
    padding: var(--padding);
    box-sizing: border-box;
}

Erklärung der einzelnen Eigenschaften

  1. flex: 1 1 calc(50% - var(--padding));

    Diese Regel setzt die Flex-Eigenschaften für die Boxen innerhalb des Flex-Containers .content.

    • flex-grow: 1;: Dies bedeutet, dass die Boxen gleichmäßig den verfügbaren Platz im Container einnehmen. Wenn zusätzlicher Platz vorhanden ist, wird dieser gleichmäßig auf alle Boxen verteilt.
    • flex-shrink: 1;: Dies bedeutet, dass die Boxen gleichmäßig schrumpfen, wenn der verfügbare Platz im Container abnimmt. Wenn der Platz knapp wird, schrumpfen alle Boxen gleichmäßig.
    • flex-basis: calc(50% - var(--padding));: Dies legt die anfängliche Größe der Boxen fest. In diesem Fall wird die Basisgröße der Boxen auf 50% der Containerbreite minus dem Wert der CSS-Variable -padding gesetzt. Die calc()Funktion wird verwendet, um diese Berechnung durchzuführen.
  1. background-color: lightgray;

    Diese Regel setzt die Hintergrundfarbe der Boxen auf lightgray.

  1. padding: var(--padding);

    Diese Regel setzt den Innenabstand (Padding) der Boxen auf den Wert der CSS-Variable --padding. Diese Variable wurde zuvor im :root-Selektor definiert:

    :root {
    --padding: 10px;
}
  1. box-sizing: border-box;

    Diese Regel ändert das Box-Modell der Boxen. Mit box-sizing: border-box; wird das Padding und der Rahmen (Border) in die Gesamtbreite und -höhe der Boxen einbezogen. Das bedeutet, dass die angegebene Breite und Höhe die gesamte Größe der Box einschließlich Padding und Border ist. Dies erleichtert das Layout, da die Boxen nicht größer werden, wenn Padding oder Border hinzugefügt werden.

Zusammengefasst

Die .content .box-Regel sorgt dafür, dass die Boxen innerhalb des Flex-Containers .content flexibel sind und sich an die verfügbare Breite anpassen. Sie haben eine anfängliche Breite von 50% des Containers minus dem Padding, eine Hintergrundfarbe von lightgray, ein Padding, das durch die CSS-Variable --padding definiert ist, und verwenden das border-box-Box-Modell, um sicherzustellen, dass Padding und Border in die Gesamtgröße der Boxen einbezogen werden.
