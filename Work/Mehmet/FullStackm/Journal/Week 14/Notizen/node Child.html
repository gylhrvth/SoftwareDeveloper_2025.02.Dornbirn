Wie importiere ich eine Funktion / Variable aus einem anderen Skript?

In Node.js und JavaScript gibt es zwei verbreitete Modul-Systeme:

  1. CommonJS (CJS) ‚Äì das klassische Node.js-Modul-System

Importieren in app.js:

2. Importieren mit ES Modules (import / export)

Hinweis: Um ES Modules in Node.js zu verwenden, setze in package.json "type": "module" oder benenne Dateien mit .mjs.

Exportieren in math.js:

  return a + b;
}

Importieren in app.js:

import { add } from './math.js';

console.log(add(2, 3)); // 5

Kurzfassung

Hinweise:

  • Beim Import unbedingt Pfadangaben wie ./ oder ../ verwenden.
  • Bei ES Modules Dateiendungen wie .js angeben.
  • Beide Module-Systeme funktionieren in Node.js, mische sie aber nicht innerhalb derselben Datei.
