HTML

Was ist HTML?

HTML (Hypertext Markup Language) ist eine Auszeichnungssprache, die zur Strukturierung und Darstellung von Inhalten im Web verwendet wird. Sie bildet das Grundgerüst einer Webseite und besteht aus einer Sammlung von Tags (Elementen), die Text, Bilder, Links, Videos und andere Inhalte auf einer Webseite definieren.

Merke: HTML ist keine Programmiersprache, sondern beschreibt nur den Aufbau einer Webseite.

Grundstrucktur einer HTML-Seite

  • "!DOCTYPE html" → Gibt an, dass es sich um HTML5 handelt.
  • "html" → Das Hauptelement, das die gesamte Seite umschließt.
  • "head" → Enthält Metadaten (z. B. den Titel der Seite).
  • "title" → Legt den Titel fest (sichtbar im Browser-Tab).
  • "body" → Enthält den sichtbaren Inhalt der Seite.
  • "h1" - "h6" → Überschriften (h1 ist die größte, h6 die kleinste).
  • "p" → Definiert einen Absatz.

Wichtige HTML-ELemente

  1. a href="" = Link
  2. img src="" = Bild einfügen
  3. ul / ol = Ungeordnete-/Geordnetelist
  4. li = Listenelement
  5. table = Tabelle
  6. div = Block-Container für Layouts
  7. span = Inline-Container für Text-Design
  8. form = Formular für Eingaben (z. B. Name, E-Mail)

HTML Attribute

HTML-Tags können Attribute haben, um zusätzliche Informationen bereitzustellen.

  • src="bild.jpg" → Gibt den Pfad zum Bild an.
  • alt="Beschreibung des Bildes" → Alternative Beschreibung (wichtig für Barrierefreiheit & SEO).

CSS für das Design hinzufügen

HTML gibt die Struktur, aber für ein schönes Design brauchst du CSS (Cascading Style Sheets).
CSS hilft dir, Farben, Schriftarten, Abstände usw. zu ändern.
