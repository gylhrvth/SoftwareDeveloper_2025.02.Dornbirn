chartAT( );

gibt das Zeichen einer bestimmten Position inerhalb eines Strings zurück

indexOf( );

sucht innerhalb eines Strings nach einem bestimmten Teilssteing und gibt den Index der ersten Übereinstimmung zurück.
Falls der gesuchte Wert nicht gefunden wird wird -1 zurückgegeben.

lastIndexOf( );

sucht innerhalb eines Strings nach einem bestimmten Teilssteing und gibt den Index der letzten Übereinstimmung zurück.

includes( );

prüft, ob eine String eine bestimmte Teilzeichenkette enthält, und gibt entweder true (gefunden) oder false (nicht gefunden) zurück.

startsWith( );

prüft, ob eine Zeichenkette mit einer bestimmten Teilzeichenkette beginnt. Sie gibt true zurück, wenn die Bedingung erfüllt ist, andernfalls false.

endssWith( );

prüft, ob eine Zeichenkette mit einer bestimmten Teilzeichenkette endet. Sie gibt true zurück, wenn die Bedingung erfüllt ist, andernfalls false.

substring( );

extrahiert einen Teilstring aus einem String, basierend auf zwei angegebenen Indizes.
Sie gibt den extrahierten Teilstring zurück, ohne den ursprünglichen String zu verändern.

slice();

Die slice()-Funktion in JavaScript extrahiert einen Teilstring aus einer Zeichenkette, basierend auf zwei angegebenen Indizes, und gibt diesen Teilstring zurück.
Im Gegensatz zu substring() kann slice() auch mit negativen Indizes arbeiten, um von hinten zu zählen.
