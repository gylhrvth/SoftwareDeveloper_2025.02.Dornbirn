UX/UI Design-User Experience

CSS

CSS (Cascading Style Sheets) ist eine Stylesheet-Sprache, die verwendet wird, um das Erscheinungsbild und Layout von HTML-Dokumenten zu gestalten. Sie ermöglicht es, Design-Elemente wie Farben, Schriftarten, Abstände und Positionierungen von HTML-Elementen festzulegen.

Eine CSS-Datei wird mit dem -Tag im -Bereich einer HTML-Datei eingebunden. Dies ermöglicht eine Trennung von Struktur (HTML) und Design (CSS).

  • Universeller Selektor (*) Wendet das Styling auf alle Elemente an.
  • Element-Selektoren (Tag-Name) Wendet das Styling auf ein bestimmtes HTML-Tag an.
  • Klassen-Selektoren (.klassenname) Wendet das Styling auf Elemente mit einer bestimmten Klasse (class) an.
  • ID-Selektoren (#idname) Wendet das Styling auf ein Element mit einer bestimmten ID (id) an.

Selectors

Selektoren in CSS werden verwendet, um bestimmte HTML-Elemente auszuwählen und zu gestalten. Sie definieren, welche Elemente ein bestimmtes Styling erhalten.

  • Element-Selektor: Stilt alle Elemente eines bestimmten Typs.
  • Klassen-Selektor (.klassenname):Stilt alle Elemente mit der angegebenen Klasse.
  • ID-Selektor (#idname):Stilt ein einzelnes Element mit der angegebenen ID.
  • ttribut-Selektor ([attribut]):Wählt Elemente mit einem bestimmten Attribut aus.
  • Pseudo-Klassen (:hover, :nth-child(), etc.):Verändern das Styling basierend auf einem Zustand.
  • Pseudo-Elemente (::before, ::after):Erstellen virtuelle Elemente vor oder nach dem Inhalt.

Das Box-Modell ist ein grundlegendes Konzept in CSS, das beschreibt, wie Elemente auf einer Webseite dargestellt werden. Jedes HTML-Element wird als eine rechteckige Box betrachtet, die aus vier Bereichen besteht:

  • Content - The content of the box, where text and images appear
  • Padding - Clears an area around the content. The padding is transparent
  • Border - A border that goes around the padding and contentMargin - Clears an area outside the border. The margin is transparent
  • Margin - Clears an area outside the border. The margin is transparent

Flex-Container

Ein Flex-Container ist ein Element in CSS, das mit display: flex; definiert wird. Er dient dazu, seine direkten Kindelemente (Flex-Items) flexibel anzuordnen.

Wann verwendet man einen Flex-Container?
Du nutzt ihn, wenn du:

  • Elemente nebeneinander oder untereinander flexibel anordnen willst.
  • Gleichmäßige Abstände zwischen Elementen brauchst.
  • Inhalte zentrieren möchtest.
  • Eine responsive Oberfläche ohne komplizierte Media Queries erstellen willst.

Flexbox ist die Abkürzung für das Flexible Box Layout-Modul.
Flexbox ist eine Layout-Methode zur Anordnung von Elementen in Reihen oder Spalten.
Flexbox erleichtert das Entwerfen einer flexiblen, responsiven Layout-Struktur, ohne float oder positioning zu verwenden.

Ein Flexbox-Layout besteht immer aus:

  • Einem Flex-Container – das übergeordnete (parent) < div >-Element
  • Flex-Items – die Elemente innerhalb des Containers (< div >)

Layout

float

Die float-Eigenschaft wird verwendet, um Inhalte zu positionieren und zu formatieren, z. B. um ein Bild links neben dem Text in einem Container schweben zu lassen.
Die float-Eigenschaft kann folgende Werte haben:

  • left – Das Element schwebt links innerhalb seines Containers.
  • right – Das Element schwebt rechts innerhalb seines Containers.
  • none – Das Element schwebt nicht (es wird genau an der Stelle im Text angezeigt, an der esvorkommt). Dies ist der Standardwert.
  • inherit – Das Element übernimmt den float-Wert seines Elternelements.

In seiner einfachsten Anwendung kann die float-Eigenschaft verwendet werden, um Text um Bilder herum fließen zu lassen.

Grid

CSS Grid Layout

Das Grid Layout-Modul bietet ein rasterbasiertes Layout-System mit Zeilen und Spalten.
Das Grid Layout-Modul ermöglicht es Entwicklern, komplexe Weblayouts einfach zu erstellen.
Das Grid Layout-Modul erleichtert das Entwerfen einer responsiven Layout-Struktur, ohne float oder positioning zu verwenden.
Die CSS-Grid-Eigenschaften werden von allen modernen Browsern unterstützt.

CSS Grid-Komponenten

Ein Grid-Layout besteht immer aus:

  • Einem Grid-Container – das übergeordnete (parent) < div >-Element
  • Grid-Items – die Elemente innerhalb des Containers (< div >)

Grid Container und Grid Items

  • Ein Grid-Layout besteht aus einem übergeordneten Element (Grid-Container) mit einem oder mehreren Grid-Items.
  • Alle direkten Kindelemente des Grid-Containers werden automatisch Grid-Items.

Z-index

Die z-index-Eigenschaft legt die Stapelreihenfolge eines Elements fest. Ein Element mit einer höheren Stapelreihenfolge steht immer vor einem Element mit einer niedrigeren Stapelreihenfolge.
Hinweis: z-index funktioniert nur bei positionierten Elementen (position: absolute;, position: relative;, position: fixed; oder position: sticky;) sowie bei Flex-Items (Elemente, die direkte Kinder eines display: flex; Containers sind).
Hinweis: Wenn zwei positionierte Elemente sich überlappen und kein z-index angegeben ist, wird das Element, das später im HTML-Code steht, oben angezeigt.

Was ist der z-index?

Der z-index in CSS bestimmt die Stapelreihenfolge von Elementen auf einer Webseite. Er legt fest, welches Element weiter vorne oder weiter hinten angezeigt wird, wenn sie sich überlappen. Wie funktioniert der z-index? Ein höherer z-index-Wert bedeutet, dass das Element weiter vorne liegt. Ein niedrigerer z-index-Wert bedeutet, dass das Element weiter hinten liegt. Standardmäßig haben alle Elemente den z-index: auto; und folgen der Reihenfolge im HTML.
Wichtig: z-index funktioniert nur, wenn das Element eine positionierte Eigenschaft hat:

  • position: relative;
  • position: absolute;
  • position: fixed;
  • position: sticky;
