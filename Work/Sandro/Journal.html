Layout Übung

Aufgaben

  1. Erstelle ein div / container

    2. Der Containter sollte in der Mitte vom Bildschirm sein und sich responsive verhalten.

    Werte:
max-width: 600px;
max-height: 300px;
  2. Dieser Container beinhaltet ein Grid

    3. Dieses Grid besteht aus zwei Reihen, erste Reihe hat zwei Elemente, die zweite Reihe hat nur ein großes Element, welches wir als Content Feld betrachten. Dieses Element wird mit Content befüllt und kann sich darum auch nach unten bzw. in der Größe erweitern. Tritt der Fall ein, dass wir über die Maximale Höhe von dem Container kommen, soll dieses Content Feld scollable werden um den Inhalt zu verabeiten.

    Werte:
max-height: 300px;  // Hierbei sollte noch beachtet werden, dass von den 300px auch noch das Padding abgezogen werden muss.
max-height: calc(300px - 40px); // Versuche zum Beispiel so einen Dynamischen Wert zu erreichen.

Beachte, dass du für jedes Element ein Layout hast um diese zu unterscheiden.
  3. Das Content feld vom Grid, beinhaltet mehrere Flex-Rows.

    4. Eine Flex Row, setzt sich aus 3 Elementen zusammen. Das erste und letzte Element, beansprucht jeweils einen(1) Teil vom platz und was als rest übrig ist, bekommt der mittlere Part. Im ersten Element wollen wir eine Checkbox, im mittleren Element einene Text und im letzten ein Bild von einem Mülleimer Icon. 

    Werte: 
height: 35px;
max-height: 35px;
