Struktur Selectors Übung

Erstelle eine Navigation Bar mit den folgenden Elementen:

  • Logo
  • Home
  • About
  • Service
  • Contact
  • Buy Now

HTML

Erstelle eine Header Section

Der Hauptcontainer ist ein div (class navbar) welche diese Elemente als Child beinhaltet.

Logo ist ein div (class logo) und wird je nach bedarf entwender ein schriftzug oder ein image

Home, About, Service, Contact sind links, welche zusammengefasst in einer UL-Liste (class links) erstellt werden

Buy Now ist ein Button (class button) und funktioniert als Link


Somit haben wir in der Navbar, 3 verschieden Elemente. Ein Logo, eine Liste, ein Button

CSS

Erstlle ein css File und verlinke es zum projekt

erstlle selctoren für folgende bereiche

  • *
  • body
  • li
  • a
  • a:hoover
  • header
  • .navbar
  • .navbar .logo
  • .navbar .links
  • .button
  • .button:hoover
  • .button:active

* selector, kümmert sich um alle. margin, padding, box-sizing (border-box) und font-family definiert!

Warum hilft es beim erstellen von einem Layout das box-sizing umzustellen?


body selector, kümmert sich um den body. height 100vh, background

Warum den Body auf 100vh einstellen?


li selector, kümmert sich um die Listen Items. list-style none

Warum list-style none in unserem Beispiel?


a selector, kümmert sich um die Links. text-decoration none, color

Warum text-decoration none in unserem Beispiel?


a:hoover selector, kümmert sich um Links wenn darüber gehoovert wird. color


header selector, kümmert sich und die section header. position relativ, padding

Warum macht es Sinn, position relativ bei einer Headersection?


.navbar selector, kümmert sich um die Navbar. display flex, justify-content space-between, align-items center, height


.navbar .logo selector, kümmert sich um das Logo. font-size, font-weight


.navbar .links selector, kümmert sich um die Links. display flex, gap,


.button selector, kümmert sich um den Button. padding, background, color, border, border-radius, etc


.button:hoover selector, kümmert sich um den Button wenn darüber gehoovert wird. background, color


.button:active selector, kümmert sich um den Button wenn er aktiviert wird. background, color
