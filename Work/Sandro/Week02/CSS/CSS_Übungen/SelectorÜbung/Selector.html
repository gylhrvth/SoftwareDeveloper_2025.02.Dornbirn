  Weißraum wird beibehalten: Alle Leerzeichen und Zeilenumbrüche im Text
  innerhalb eines pre-Elements werden beibehalten und genauso angezeigt, wie sie
  im HTML-Dokument geschrieben sind. Link

  Monospace-Schriftart: Der Text innerhalb eines pre-Elements wird standardmäßig
  in einer Monospace-Schriftart (auch als nicht-proportionale Schriftart bekannt)
  angezeigt. Das bedeutet, dass jedes Zeichen die gleiche Breite hat, was es
  einfacher macht, Text in Spalten auszurichten.
