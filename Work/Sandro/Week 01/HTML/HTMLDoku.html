Frage 1

Was ist HTML und wofür wird es verwendet?

HTML (HyperText Markup Language) ist die Standard-Auszeichnungssprache für Dokumente, die im Web angezeigt werden sollen. Es wird verwendet, um die Struktur und den Inhalt einer Webseite zu beschreiben, indem es Elemente wie Überschriften, Absätze, Links, Bilder und andere Multimedia-Inhalte definiert.

Frage 2

Welche grundlegenden HTML-Tags gibt es und wofür werden sie verwendet?

Zu den grundlegenden HTML-Tags gehören:

    Essentielle Elemente von HTML

    Die essentiellen Elemente von HTML sind grundlegende Bausteine, die für die Erstellung eines HTML-Dokuments erforderlich sind. Diese Elemente umfassen:

    • <!DOCTYPE> - Deklariert den Dokumenttyp und die verwendete HTML-Version.
    • <html> - Das Wurzelelement des HTML-Dokuments.
    • <head> - Enthält Metadaten über das Dokument.
    • <title> - Definiert den Titel des Dokuments, der in der Titelleiste des Browsers angezeigt wird.
    • <meta> - Definiert Metadaten wie Zeichensatz, Autor, Beschreibung usw.
    • <link> - Verknüpft externe Ressourcen wie CSS-Dateien.
    • <script> - Fügt JavaScript in das Dokument ein.
    • <body> - Enthält den sichtbaren Inhalt des Dokuments.

    Weitere wichtige HTML-Tags

  • <h1> bis <h6> - Definiert Überschriften, wobei <h1> die wichtigste und <h6> die am wenigsten wichtige ist.
  • <p> - Definiert einen Absatz.
  • <a> - Definiert einen Hyperlink.
  • <img> - Fügt ein Bild ein.
  • <ul> - Definiert eine ungeordnete Liste.
  • <ol> - Definiert eine geordnete Liste.
  • <li> - Definiert ein Listenelement.
  • <div> - Definiert einen Abschnitt oder ein Container-Element.
  • <span> - Definiert einen Inline-Bereich im Text.
  • <pre> - Definiert einen vorformatierten Textblock.
  • <button> - Definiert eine Schaltfläche.
  • <input> - Definiert ein Eingabefeld, das Benutzereingaben ermöglicht. Es kann verschiedene Typen wie Text, Passwort, E-Mail usw. haben.

Textformatierungselemente

Hier sind einige der am häufigsten verwendeten HTML-Elemente zur Textformatierung:

  • <strong> - Hebt Text hervor, indem er fett dargestellt wird. Es signalisiert auch eine starke Betonung.
  • <em> - Hebt Text hervor, indem er kursiv dargestellt wird. Es signalisiert auch eine Betonung.
  • <b> - Macht Text fett, ohne zusätzliche Betonung.
  • <i> - Macht Text kursiv, ohne zusätzliche Betonung.
  • <u> - Unterstreicht Text.
  • <mark> - Hebt Text hervor, indem er markiert wird (normalerweise gelb hinterlegt).
  • <del> - Stellt durchgestrichenen Text dar, um gelöschten Text anzuzeigen.
  • <code> - Stellt Text in einer monospace-Schriftart dar, um Code zu kennzeichnen.

Veraltete HTML-Elemente

Einige HTML-Elemente werden heutzutage nicht mehr empfohlen, da sie veraltet sind und durch modernere, flexiblere Techniken ersetzt wurden. Hier sind einige Beispiele:

  • <table> für Layouts - Tabellen wurden früher häufig für das Seitenlayout verwendet. Dies ist jedoch nicht mehr empfehlenswert, da Tabellen nicht responsiv sind und das Layout bei verschiedenen Bildschirmgrößen nicht gut anpassen. Stattdessen sollten CSS-Layouts wie Flexbox oder Grid verwendet werden.
  • <font> - Dieses Tag wurde verwendet, um Textstil und -farbe zu definieren. Es ist veraltet, da CSS eine viel bessere und flexiblere Möglichkeit bietet, das Aussehen von Text zu steuern.
  • <center> - Dieses Tag wurde verwendet, um Inhalte zu zentrieren. Es ist veraltet, da CSS-Methoden wie text-align: center; oder Flexbox verwendet werden sollten.
  • <marquee> - Dieses Tag wurde verwendet, um Text oder Bilder horizontal oder vertikal scrollen zu lassen. Es ist veraltet und wird von modernen Browsern nicht mehr unterstützt. Stattdessen sollten CSS-Animationen verwendet werden.
  • <u> - Dieses Tag wurde verwendet, um Text zu unterstreichen. Es ist veraltet, da CSS eine bessere Kontrolle über Textdekorationen bietet.
  • <big> und <small> - Diese Tags wurden verwendet, um die Schriftgröße zu ändern. Sie sind veraltet, da CSS eine präzisere Kontrolle über die Schriftgröße bietet.
  • <strike> - Dieses Tag wurde verwendet, um durchgestrichenen Text darzustellen. Es ist veraltet und sollte durch <del> oder CSS ersetzt werden.
