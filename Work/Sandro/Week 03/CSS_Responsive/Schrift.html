Ich bin eine Überschrift

Für Schriftgrößen, gibt es viele Möglichkeiten die größe zu definiern.

  • px - Pixel
  • em - Elternelements basiert.
  • rem - Root Em: Wurzelelements (html) basiert.
  • pt - Punkt (1 Punkt = 1/72 Zoll).
  • pc - Pica (1 Pica = 12 Punkte).
  • vw - Viewport Width 1% der Breite des Ansichtsfensters entspricht.
