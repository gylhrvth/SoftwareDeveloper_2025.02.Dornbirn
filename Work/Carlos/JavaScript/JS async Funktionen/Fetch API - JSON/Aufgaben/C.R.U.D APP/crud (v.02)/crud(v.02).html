CRUD (V.01)

Einführung

CRUD steht für Create, Read, Update und Delete. Es handelt sich um die vier grundlegenden Operationen, die auf Datenbanken angewendet werden.

In dieser Anwendung kann man Objekte erstellen, lesen, aktualisieren und löschen.

Die Anwendung verwendet die Fetch API, um mit einer JSON-Datenbank zu kommunizieren.

Die Datenbank ist eine einfache JSON-Datei, die auf dem Server gespeichert ist.

Die Anwendung verwendet auch die async/await-Syntax, um asynchrone Operationen zu handhaben.
