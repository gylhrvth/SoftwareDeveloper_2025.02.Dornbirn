Dienstag, 18. März 2025

JavaScript (Datatypes)

  • Was sind die 7 + 1 standard JS Datatypes?

    • In JavaScript gibt es primitive und nicht-primitive (Referenz-) Datentypen.

    • Primitive Datentypen (immutable, werden direkt im Speicher gespeichert):

      • string: let name = "Carlos";
      • number: let age = 35;
      • boolean: let isDeveloper = true;
      • undefined: Eine Variable, die deklariert, aber nicht initialisiert wurde -> let notAssigned;
      • null: Bewusste leere -> let emptyValue = null;
      • bigint: Große ganze Zahlen Nummer endet mit "n"-> let bigNumber = 9007199254740991n;
      • symbol: Einzigartige Identifikatoren -> let uniqueId = Symbol("id");

    • Complex Datentypen (verweisen auf Speicheradressen):

      • Object: Sammlungen von Werten -> let person = { name: "Carlos", age: 35 };

  • Was passiert wenn man verschiedene Datatypen zusammen verwendet?

    • Das Verhalten hängt davon ab, welche Typen man kombiniert. Das nennt man Type Coercion (Typumwandlung). JavaScript wandelt dabei automatisch Werte in kompatible Typen um.

    • Bei additionen: String + Number → String -> console.log("10" + 5); // "105"

    • Bei anderen Operatoren: Hier erkennt JavaScript, dass die Strings eigentlich Zahlen sind, und wandelt sie in number um -> console.log("10" - 5); // 5 (String wird in Number umgewandelt)

    • Boolean + Number: console.log(true + 1); // 2 (true wird zu 1) / console.log(false + 1); // 1 (false wird zu 0)

    • Null: console.log(null + 5); // 5 (null wird zu 0)

    • Undefined: console.log(undefined + 5); // NaN (not a number) (undefined kann nicht in eine Zahl umgewandelt werden)

  • Beispiel: Was passiert wenn man 10 + "Apfel" schreibt?

    • Man bekommt als Ergebnis einen string: "10Apfel". Das passiert wenn man ein Number mit einem String kombiniert.

  • Beispiel: Was passiert wenn man 0.1 + 0.2 schreibt?

    • Man bekommt als Ergebnis 0.30000000000000004. Das liegt an der IEEE 754 Gleitkommazahlen-Darstellung, die JavaScript (und viele andere Sprachen) für number-Werte verwendet.

    0.1 und 0.2 haben keine exakte binäre Darstellung, ähnlich wie 1/3 in Dezimalsystem unendlich viele Nachkommastellen hat (0.33333…).

    Dadurch entstehen kleine Rundungsfehler bei Berechnungen.

  • Was macht die toFixed() Funktion? Wofür ist die geeignet?

    • Die Funktion toFixed(n) rundet auf "n" Nachkommastellen, gibt aber einen String zurück.

    Beispiel: console.log((0.1 + 0.2).toFixed(2)); // "0.30" (String!)

    Number(...) konvertiert den String zurück in eine Zahl. Beispiel: console.log(Number((0.1 + 0.2).toFixed(2))); // 0.3 (Number)

  • Was macht die toString() Funktion? Wofür ist die geeignet?

    • Die toString()-Funktion in JavaScript wandelt eine Zahl, ein Objekt oder einen anderen Wert in einen String um.

    Beispiel: let num = 42; -> console.log(num.toString()); // "42"

  • Was macht die valueOf() Funktion? Wofür ist die geeignet?

    • Die Methode valueOf() gibt den “echten” Wert eines Objekts zurück.

    • Bei Zahlen, Strings oder Booleans: valueOf() gibt einfach den Wert selbst zurück. Beispiel: console.log(("Hallo").valueOf()); // "Hallo"

    • Bei Objekten (Standardverhalten): let obj = { a: 10 }; console.log(obj.valueOf()); // { a: 10 } -> Hier passiert nichts Besonderes – das Objekt bleibt ein Objekt.

    • valueOf() wird nützlich, wenn man ein Objekt in eine Zahl oder einen String umwandeln will. Beispiel: let date = new Date("2024-03-18"); -> console.log(date.valueOf()); // 1710705600000 (Zeit in Millisekunden)

  • Was ist der typeof Operator? Wofür ist der nützlich?

    • Der typeof Operator gibt den Datentyp eines Werts in JavaScript als String zurück. Beispiele: console.log(typeof 42);// "number"

    Es ist sehr nützlich für Typ-Prüfungen und Fehlervermeidung.

Donnerstag, 13. März 2025

JavaScript (Basics)

JS Grundlagen

  • Was ist JavaScript?

    • JavaScript ist eine Programmiersprache, die hauptsächlich für interaktive Webseiten genutzt wird. Sie läuft im Browser und ermöglicht dynamische Inhalte wie Animationen, Formulare oder interaktive Elemente.

  • Was kann man mit Java Script in verbindung mit Web-Development machen?

    • Interaktive Elemente erstellen (Buttons, Menüs, Formulare).

    • Animationen & Effekte umsetzen (z. B. mit CSS oder Canvas).

    • DOM-Manipulation durchführen (Ändern von HTML und CSS über JS).

    • Benutzereingaben validieren (z. B. Formulareingaben prüfen).

    • Dynamische Inhalte laden und aktualisieren (z. B. ohne die Seite neu zu laden – AJAX, Fetch API).

    • Daten speichern & verwalten (z. B. mit Local Storage oder APIs).

    • Frameworks & Bibliotheken nutzen (z. B. React, Vue, jQuery).

  • Welche Möglichkeiten gibt es Java Script Code zu schreiben und in ein Projekt einzufügen?

    • Es gibt mehrere Möglichkeiten, JavaScript-Code zu schreiben und in ein Webprojekt einzufügen:

    • Direkt im HTML-Dokument (Inline-Skript)

      • <button onclick="alert('Hallo!')">Klick mich</button>

    • Im <script>-Tag innerhalb von HTML

      • <script> console.log("Hallo, Welt!"); </script>

    • Externe JavaScript-Datei (.js-Datei)

      • <script src="script.js"></script>

    • Modul-System (type="module")

      • <script type="module" src="script.js"></script>

    • JavaScript über Node.js (Serverseitig)

    Für größere Projekte ist eine externe .js-Datei mit Modul-System am besten, um den Code übersichtlich und wartbar zu halten.

  • Wie schreibt man in Java Script Kommentare?

    • In JavaScript gibt es zwei Arten von Kommentaren:

    • Einzeiliger Kommentar (//)

      • // Das ist ein einzeiliger Kommentar console.log("Hallo, Welt!"); // Kommentar nach einer Codezeile

    • Mehrzeiliger Kommentar (/* ... */)

      • /* Das ist ein mehrzeiliger Kommentar. Er kann über mehrere Zeilen gehen. */ console.log("Hallo, Welt!");

  • Wenn man Namen definiert für Variablen oder Funktionen, welche Regeln muss man beachten?

    • ✅ Erlaubt:

      • Buchstaben (a-z, A-Z)

      • Ziffern (0-9), aber nicht am Anfang

      • _ (Unterstrich) und $ (Dollarzeichen)

      • Groß- und Kleinschreibung wird unterschieden (name ≠ Name)

    • ❌ Nicht erlaubt:

      • Namen dürfen nicht mit einer Zahl beginnen (1variable ❌)

      • Keine Leerzeichen oder Sonderzeichen außer _ und $

      • Reservierte Wörter (z. B. let, function, return) dürfen nicht als Namen verwendet werden


    JS Programming Basics

    • Was sind sogenannte Statements?

      • Ein Statement (Anweisung) in JavaScript ist eine einzelne ausführbare Codezeile, die eine bestimmte Aktion ausführt. (z. B. eine Variable setzen, eine Bedingung prüfen oder eine Schleife ausführen).

      let name = "Carlos"; // Variablenzuweisung console.log(name); // Funktionsaufruf if (name === "Carlos") { // Bedingung console.log("Hallo, Carlos!"); }

    • Was bedeutet Syntax?

      • Syntax bezeichnet die Regeln und Struktur einer Programmiersprache, die festlegen, wie Code korrekt geschrieben werden muss, damit der Computer ihn versteht und ausführt.

    • Was sind Variablen?

      • Variablen sind Platzhalter für Daten in einem Programm. Sie ermöglichen es, Werte zu speichern und später im Code zu verwenden. In JavaScript sind Variablen flexible Behälter, die unterschiedliche Datentypen aufnehmen können (z. B. Zahlen, Texte, Wahrheitswerte).

      Werte können unterschiedlich sein, z. B. Zahlen (30), Strings ("Hallo"), Booleans (true/false), Arrays oder Objekte.

      let name = "Carlos"; // Variablenname: name, Wert: "Carlos" let age = 30; // Variablenname: age, Wert: 30

    • Was sind Booleans?

      • Booleans sind ein Datentyp in JavaScript, der nur zwei mögliche Werte haben kann: true (wahr) oder false (falsch). Sie werden häufig in Bedingungen, Vergleichen und logischen Operationen verwendet.

    • Was sind Operatoren?

      • Operatoren sind spezielle Zeichen, die bestimmte Aktionen wie Berechnungen, Vergleiche oder logische Operationen auf Variablen oder Werten ausführen.

    • Was ist eine Funktion?

      • Eine Funktion in JavaScript ist ein wiederverwendbarer Codeblock, der eine bestimmte Aufgabe oder Berechnung ausführt. Funktionen ermöglichen es, komplexe Aufgaben in kleinere, handhabbare Teile zu unterteilen und den Code effizienter und übersichtlicher zu gestalten.

      function greet() { console.log("Hallo, Welt!"); } greet(); // Ruft die Funktion auf und gibt "Hallo, Welt!" aus

    • Was ist eine Schleife (Loop)?

      • Eine Schleife in JavaScript ist eine Kontrollstruktur, die es ermöglicht, einen bestimmten Codeblock wiederholt auszuführen, solange eine Bedingung erfüllt ist. Schleifen sind nützlich, wenn man eine Aktion mehrmals durchführen möchte, ohne den gleichen Code mehrfach zu schreiben.

      Es gibt verschiedene Arten von Schleifen (z. B. for, while, do...while), die sich durch ihre Struktur und die Art der Bedingungsprüfung unterscheiden.

    • Was ist das If-Else Statement?

      • Das if-else-Statement in JavaScript ist eine Bedingungsanweisung, die es ermöglicht, unterschiedliche Codeblöcke auszuführen, je nachdem, ob eine bestimmte Bedingung wahr (true) oder falsch (false) ist.
