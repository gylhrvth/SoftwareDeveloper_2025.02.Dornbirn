JavaScript (Event Handling)

  • Was ist ein Event?

    • Ein Event ist in JavaScript ein Signal, das vom Browser oder einem Benutzer ausgelöst wird – zum Beispiel:

    • click – jemand klickt auf ein Element
    • mouseover – Maus bewegt sich über ein Element
    • keydown - eine Taste wird gedrückt
    • submit - ein Formular wird abgeschickt
    • change - Eingabefeld ändert sich
    • input - Nutzer tippt in ein Textfeld
    • resize – Fenstergröße wird geändert
    • scroll – Nutzer scrollt auf der Seite

  • Was ist ein (Event) Handler?

    • Ein Event Handler (auch Event Listener genannt) ist eine Funktion, die auf ein bestimmtes Ereignis (Event) reagiert. Der Event Handler wird auf ein Element „gesetzt“ und wird ausgeführt, wenn das Ereignis eintritt, wie zum Beispiel ein Klick, eine Tastenbetätigung oder eine Mausbewegung.

    Beispiel:

    <button id="meineTaste">Klick mich!</button> <script> const button = document.getElementById("meineTaste"); // Event Handler für das 'click'-Event button.addEventListener("click", function() { alert("Der Button wurde geklickt!"); }); </script>

  • Wie fügt man statisch ein Event Handler zu einen HTML Element?

    • Um statisch einen Event Handler zu einem HTML-Element hinzuzufügen, kann man onclick, onmouseover, onchange, oder ein anderes Event-Attribut direkt im HTML-Tag verwenden. Dies wird als statische Event-Zuordnung bezeichnet, da der Event Handler direkt im HTML definiert wird.

    Beispiel mit "onclick":

    <button onclick="zeigeNachricht()">Klick mich!</button> <script> function zeigeNachricht() { alert("Der Button wurde geklickt!"); } </script>

  • Wie fügt man dynamisch ein Event Handler zu einen HTML Element?

    • Um dynamisch einen Event Handler zu einem HTML-Element hinzuzufügen, verwendet man die Methode addEventListener() in JavaScript. Diese Methode ermöglicht es, ein Event und eine zugehörige Funktion (den Event Handler) zu einem Element hinzuzufügen.

    Beispiel:

    <button id="meineTaste">Klick mich!</button> <script> const button = document.getElementById("meineTaste"); // Event Handler für das 'click'-Event button.addEventListener("click", function() { alert("Der Button wurde geklickt!"); }); </script>

    In diesem Beispiel wird der Event Handler für das 'click'-Event dynamisch hinzugefügt, wenn die Seite geladen wird. Wenn der Button geklickt wird, wird die Funktion ausgeführt und eine Nachricht angezeigt.


  • Was ist der Unterschied zwischen addEventListener und onclick?

    • Der Hauptunterschied zwischen addEventListener und onclick ist:

    • addEventListener ermöglicht es, mehrere Event Handler für dasselbe Event auf demselben Element hinzuzufügen. onclick hingegen überschreibt jeden vorherigen Event Handler, der für dasselbe Event auf demselben Element gesetzt wurde.

    • addEventListener bietet mehr Flexibilität und Kontrolle über die Ereignisbehandlung, während onclick einfacher zu verwenden ist, aber weniger Möglichkeiten bietet.

    • addEventListener kann auch für andere Ereignisse als nur "click" verwendet werden, während onclick speziell für das "click"-Ereignis gedacht ist.

    • addEventListener ermöglicht die Verwendung von Optionen wie "capture" und "once", während onclick diese Optionen nicht unterstützt.

  • Was ist ein Page Load Event?

    • Das Page Load Event ist ein Ereignis, das ausgelöst wird, wenn eine Webseite vollständig geladen ist. Dies bedeutet, dass alle HTML-Elemente, CSS-Stile und JavaScript-Skripte auf der Seite vollständig geladen und bereit sind.

    Man sollte DOMContentLoaded verwenden, wenn man nur auf das Laden des HTML-Dokuments wartet.

    Man sollte load verwenden, wenn man sicherstellen möchte, dass alle Ressourcen (wie Bilder) geladen sind.

    Beispiel:

    <script> window.addEventListener("load", function() { console.log("Die Seite ist vollständig geladen!"); }); </script>

    In diesem Beispiel wird eine Funktion registriert, die ausgeführt wird, wenn die Seite vollständig geladen ist. In diesem Fall wird eine Nachricht in der Konsole angezeigt.


  • Was ist eine Mouse Event?

    • Ein Mouse Event ist ein Ereignis, das auftritt, wenn der Benutzer mit der Maus interagiert. Es gibt verschiedene Arten von Mouse Events, die in JavaScript verwendet werden können, um auf Mausaktionen zu reagieren.

    Beispiele für Mouse Events sind:

    • click – wenn der Benutzer auf ein Element klickt
    • dblclick – wenn der Benutzer doppelt auf ein Element klickt
    • mouseover – wenn der Mauszeiger über ein Element bewegt wird
    • mouseout – wenn der Mauszeiger das Element verlässt
    • mousemove – wenn sich die Maus über ein Element bewegt
    • mousedown – wenn eine Maustaste gedrückt wird
    • mouseup – wenn eine Maustaste losgelassen wird
    • wheel – wenn das Mausrad gedreht wird
    • contextmenu – wenn das Kontextmenü (Rechtsklick) geöffnet wird

    Beispiel:

    <div id="meineBox" style="width: 200px; height: 200px; background-color: lightblue;"> </div> <script> const box = document.getElementById("meineBox"); // Mouseover Event box.addEventListener("mouseover", function() { box.style.backgroundColor = "lightgreen"; }); // Mouseout Event box.addEventListener("mouseout", function() { box.style.backgroundColor = "lightblue"; }); </script>

    In diesem Beispiel wird die Hintergrundfarbe der Box geändert, wenn der Mauszeiger darüber bewegt wird (mouseover) und zurückgesetzt, wenn der Mauszeiger die Box verlässt (mouseout).


  • Was ist ein Keyboard Event?

    • Ein Keyboard Event ist ein Ereignis, das auftritt, wenn der Benutzer mit der Tastatur interagiert. Es gibt verschiedene Arten von Keyboard Events, die in JavaScript verwendet werden können, um auf Tasteneingaben zu reagieren.

    Beispiele für Keyboard Events sind:

    • keydown – wenn eine Taste gedrückt wird
    • keyup – wenn eine Taste losgelassen wird
    • keypress – wenn eine Taste gedrückt wird (veraltet, nicht mehr empfohlen)

    Beispiel:

    <input type="text" id="textInput"> <script> const input = document.getElementById("textInput"); // Keydown Event input.addEventListener("keydown", function(event) { console.log("Taste gedrückt: " + event.key); }); // Keyup Event input.addEventListener("keyup", function(event) { console.log("Taste losgelassen: " + event.key); }); </script>

    In diesem Beispiel wird eine Nachricht in der Konsole angezeigt, wenn eine Taste gedrückt oder losgelassen wird, während der Benutzer in das Eingabefeld tippt.


  • Woher weiß man welche Taste gedruckt wurde?

    • In einem Keyboard Event, mit der Eigenschaft event.key.

    Beispiel:

    <input type="text" id="textInput"> <script> const input = document.getElementById("textInput"); // Keydown Event input.addEventListener("keydown", function(event) { console.log("Taste gedrückt: " + event.key); }); </script>

    In diesem Beispiel wird die gedrückte Taste in der Konsole angezeigt, wenn der Benutzer eine Taste drückt, während er in das Eingabefeld tippt.

    Die function(event) ist eine anonyme Funktion, die als Event-Handler verwendet wird. Das event ist ein Parameter, der ein Event-Objekt darstellt. Dieses Objekt enthält Informationen über das Ereignis, das ausgelöst wurde, wie z. B. welche Taste gedrückt wurde, welches Element geklickt wurde oder welche Mausbewegung stattgefunden hat.

    Diese Funktion wird ausgeführt, wenn das keydown-Ereignis ausgelöst wird (also wenn der Benutzer eine Taste drückt, während das Eingabefeld aktiv ist).


  • Was ist ein Form submit?

    • Ein Form Submit ist ein Ereignis, das auftritt, wenn ein Formular abgeschickt wird. Dies geschieht normalerweise, wenn der Benutzer auf einen "Submit"-Button klickt oder die Eingabetaste drückt, während sich der Fokus im Formular befindet.

    Beispiel:

    <form id="meinFormular"> <input type="text" id="name" placeholder="Name"> <input type="email" id="email" placeholder="E-Mail"> <button type="submit">Absenden</button> </form> <script> const formular = document.getElementById("meinFormular"); formular.addEventListener("submit", function(event) { event.preventDefault(); // Verhindert das Standardverhalten des Formulars console.log("Formular abgeschickt!"); }); </script>

    In diesem Beispiel wird eine Nachricht in der Konsole angezeigt, wenn das Formular abgeschickt wird. Das event.preventDefault() verhindert, dass die Seite neu geladen wird, was normalerweise passiert, wenn ein Formular abgeschickt wird.


  • Was ist die preventDefault() Funktion?

    • Die preventDefault() Funktion ist eine Methode des Event-Objekts in JavaScript, die verwendet wird, um das Standardverhalten eines Ereignisses zu verhindern. Wenn diese Methode aufgerufen wird, wird das Standardverhalten des Browsers für dieses Ereignis nicht ausgeführt.

    Beispiel:

    const formular = document.getElementById("meinFormular"); formular.addEventListener("submit", function(event) { event.preventDefault(); // Verhindert das Standardverhalten const name = document.getElementById("name").value; const email = document.getElementById("email").value; if (name.trim() === "" || email.trim() === "") { console.log("Bitte füllen Sie alle Felder aus."); } else { console.log("Formular erfolgreich abgeschickt!"); // Hier könntest du die Daten manuell an den Server senden } });

    Es wird überprüft, ob die Felder "Name" und "E-Mail" ausgefüllt sind. Die Nachricht "Formular erfolgreich abgeschickt!" wird nur angezeigt, wenn die Validierung erfolgreich ist.

    Das event.preventDefault() verhindert nur das Standardverhalten des Browsers, nicht die Ausführung des Event-Handlers.

JavaScript (DOM)

  • Wo für steht die Abkürzung DOM?

    • Die Abkürzung DOM steht für: Document Object Model.

    Das DOM ist eine Programmierschnittstelle (API), mit der man auf HTML- und XML-Dokumente zugreifen und sie verändern kann – z. B. mit JavaScript.

    Das DOM ist der Weg, wie JavaScript mit einer Webseite interagiert.

  • Was kann man mit dem DOM bearbeiten?

    • Mit dem DOM kann man z. B.:

    • Elemente hinzufügen/löschen (createElement, removeChild)

    • Inhalte ändern (innerText, textContent)

    • Stile verändern (element.style)

    • Events anhängen (addEventListener)

  • Was kann ich mit dem DOM nicht bearbeiten?

    • Auf Dateien/Festplatte zugreifen: Man kann z.B nicht auf Dateien des Nutzers zugreifen, Programme starten oder Betriebssystembefehle ausführen.

    • Auf fremde Webseiten zugreifen: Der Browser blockiert das aus Sicherheitsgründen (Cross-Origin Restriction).

    • Rechenintensive Aufgaben effizient lösen: Besser reines JavaScript verwenden.

    • Bilder direkt bearbeiten: Keine native Grafikbearbeitung.

  • Wie kann ich ein HTML Element mit einem eindeutigen ID erreichen?

    • Mit der Methode -> document.getElementById("ID")

    Beispiel (html):

    <h1 id="headline">Willkommen!</h1> <button onclick="changeText()">Klick mich</button>

    Beispiel (js):

    function changeText() { let element = document.getElementById("headline") element.innerText = "Hallo vom DOM!" }

  • Wie kann ich alle HTML Elementen erreichen, die eine class="xyz" haben?

    • Mit der Methode -> document.getElementsByClassName("xyz")

    Beispiel (html):

    <p class="xyz">Text 1</p> <p class="xyz">Text 2</p> <div class="xyz">Ein div</div>

    Beispiel (js):

    let elements = document.getElementsByClassName("xyz") for (let el of elements) { el.style.color = "blue" }

    Das macht alle Elemente mit der Klasse xyz blau.

  • Wie kann man alle HTML Elementen erreichen, die man mit einem CSS Selector beschreiben kann?

    • Mit der Methode -> document.querySelectorAll(".xyz")

    Beispiel (html):

    <div class="box">Box 1</div> <div class="box">Box 2</div> <div class="box">Box 3</div> <button onclick="changeBoxes()">Ändere alle Boxen</button>

    Beispiel (js):

    function changeBoxes() { // Alle Elemente mit der Klasse "box" auswählen let boxes = document.querySelectorAll(".box") // Durch alle ausgewählten Elemente iterieren boxes.forEach(box => { box.style.backgroundColor = "lightgreen" // Hintergrundfarbe ändern box.style.padding = "20px" // Padding ändern box.innerText = "Die Box wurde geändert!" // Text ändern }) }

  • Wie kann man aus einem Element der Parent-Element erreichen?

    • Mit der Eigenschaft parentElement.

    Beispiel (html):

    <div id="parent"> <p id="child">Ich bin ein Kind-Element.</p> </div> <button onclick="findParent()">Finde das Elternelement</button>

    Beispiel (js):

    function findParent() { let child = document.getElementById("child") let parent = child.parentElement // Parent-Element des Kindes holen console.log(parent) // Gibt das Elternelement im Console aus parent.style.backgroundColor = "lightblue" // Optional: Hintergrundfarbe des Eltern-Elements ändern }

    child.parentElement: Diese Eigenschaft gibt das Elternelement des child-Elements zurück.

    Wenn man auf den Button klickt, wird die Konsole das Elternelement ausgeben und dessen Hintergrundfarbe ändern.

  • Wie kann man alle Child-Elements eines Elements erreichen?

    • Um alle Child-Elemente eines bestimmten Elements zu erreichen, kann man die Eigenschaft "children" verwenden. Diese gibt eine HTMLCollection aller direkten Kinder eines Elements zurück.

    Beispiel (html):

    <div id="parent"> <p>Kind 1</p> <p>Kind 2</p> <p>Kind 3</p> </div> <button onclick="changeChildren()">Ändere alle Kind-Elemente</button>

    Beispiel (js):

    function changeChildren() { let parent = document.getElementById("parent") // Elternelement holen for (let child of parent.children) { child.style.color = "blue" // Alle Kind-Elemente blau färben } }

    Wenn man auf den Button klickt, wird die Farbe jedes Kind-Elements (die p-Tags) blau gesetzt.

  • Wie kann man den Text Content eines Elements ändern?

    • Um den Text-Inhalt eines Elements zu ändern, kannst man die Eigenschaft textContent verwenden. Mit textContent kannst man den gesamten Text eines Elements abrufen oder ändern.

    Beispiel (html):

    <p id="textElement">Dieser Text wird geändert.</p> <button onclick="changeText()">Ändere den Text</button>

    Beispiel (js):

    function changeText() { let element = document.getElementById("textElement"); // Das Element holen element.textContent = "Der Text wurde geändert!"; // Den Text ändern }

  • Wie kann man die Attribute eines Elements ändern?

    • Um die Attribute eines Elements zu ändern, kann man die Methoden setAttribute oder direkt die Eigenschaften des spezifischen Attributs verwenden (wie src für ein Bild oder href für einen Link).

    -> setAttribute Methode Beispiel (html):

    <img id="image" src="old-image.jpg" alt="Altes Bild" width="200"> <button onclick="changeImage()">Bild ändern</button>

    -> setAttribute Methode Beispiel (js):

    function changeImage() { let img = document.getElementById("image"); // Das Bild-Element holen img.setAttribute("src", "new-image.jpg"); // Das src-Attribut ändern img.setAttribute("alt", "Neues Bild"); // Das alt-Attribut ändern }

    Für einige Standard-Attribute kann man auch direkt die entsprechende Eigenschaft des Elements verwenden (wie src für Bilder, href für Links):

    -> Direkte Eigenschaft verwenden (html):

    <img id="image" src="old-image.jpg" alt="Old Image" width="200"> <button onclick="changeImage()">Change Image</button>

    -> Direkte Eigenschaft verwenden (js):

    function changeImage() { let img = document.getElementById("image"); // Get the image element img.src = "new-image.jpg"; // Change the src attribute to a new image img.alt = "New Image"; // Optionally, change the alt attribute as well }

  • Wie kann man die Class Liste eines Elements erreichen?

    • Man kann die Class-Liste eines Elements in JavaScript mit der Eigenschaft .classList erreichen. Das ist ein praktisches Objekt, mit dem man Klassen hinzufügen, entfernen, toggeln oder überprüfen kann.

    Beispiel:

    <p id="text" class="blau">Hallo Welt!</p> <script> const element = document.getElementById("text"); // Zeigt alle Klassen im Element console.log(element.classList); // Ausgabe: ['blau'] // Eine neue Klasse hinzufügen element.classList.add("fett"); // Eine vorhandene Klasse entfernen element.classList.remove("blau"); </script>

  • Was macht classList.add('abc') Funktion?

    • Die Funktion classList.add('abc') fügt dem HTML-Element eine neue CSS-Klasse namens abc hinzu, falls sie noch nicht vorhanden ist.

    <p id="text" class="rot">Hallo!</p> <script> const element = document.getElementById("text"); element.classList.add("fett"); console.log(element.className); // Ausgabe: "rot fett" </script>

  • Was macht classList.remove('abc') Funktion?

    • Die Funktion classList.remove('abc') entfernt die CSS-Klasse abc von einem HTML-Element — aber nur, wenn sie vorhanden ist.

    Beispiel:

    <p id="text" class="rot fett">Hallo!</p> <script> const element = document.getElementById("text"); element.classList.remove("rot"); console.log(element.className); // Ausgabe: "fett" </script>

  • Was macht classList.toggle('abc') Funktion?

    • Die Funktion classList.toggle('abc') fügt die Klasse abc hinzu, wenn sie nicht vorhanden ist – und entfernt sie, wenn sie bereits vorhanden ist. Es ist also ein Ein/Aus-Schalter für Klassen.

    Beispiel:

    <p id="text">Hallo!</p> <button onclick="toggleFarbe()">Farbe wechseln</button> <script> function toggleFarbe() { const element = document.getElementById("text"); element.classList.toggle("rot"); } </script> <style> .rot { color: red; } </style>

  • Wie erstellt man ein neues HTML Element?

    • Um ein neues HTML-Element mit JavaScript zu erstellen, verwendet man die Methode document.createElement().

    Beispiel (js):

    // Neues <p>-Element erstellen const neuerAbsatz = document.createElement("p"); // Textinhalt hinzufügen neuerAbsatz.textContent = "Das ist ein neuer Absatz."; // Element in den DOM einfügen, z. B. in den Body document.body.appendChild(neuerAbsatz);

  • Wie kann man ein Element an eine bestimmte Position im HTML-Dokument einfügen?

    • Die gängigsten Methoden sind:

    • appendChild() → ans Ende eines Elements anhängen

      • elternElement.appendChild(neuesElement);

    • prepend() → an den Anfang eines Elements anhängen

      • elternElement.prepend(neuesElement);

    • insertBefore() → vor einem bestimmten Element einfügen

      • elternElement.insertBefore(neuesElement, referenzElement);

    • insertAdjacentElement(position, element) → an eine genaue Position

      • beforebegin: direkt vor dem #zielElement.
      • afterbegin: ganz oben innerhalb von #zielElement.
      • beforeend: ganz unten innerhalb von #zielElement
      • afterend: direkt nach dem #zielElement

      Beispiel (Ausgangs-HTML):

      <div id="box"> <p>Inhalt der Box</p> </div>

      Beispiel (JavaScript):

      const ziel = document.getElementById("box"); const beforeBegin = document.createElement("div"); beforeBegin.textContent = "👉 beforebegin"; beforeBegin.style.color = "blue"; const afterBegin = document.createElement("div"); afterBegin.textContent = "👉 afterbegin"; afterBegin.style.color = "green"; const beforeEnd = document.createElement("div"); beforeEnd.textContent = "👉 beforeend"; beforeEnd.style.color = "orange"; const afterEnd = document.createElement("div"); afterEnd.textContent = "👉 afterend"; afterEnd.style.color = "red"; // Elemente einfügen ziel.insertAdjacentElement("beforebegin", beforeBegin); ziel.insertAdjacentElement("afterbegin", afterBegin); ziel.insertAdjacentElement("beforeend", beforeEnd); ziel.insertAdjacentElement("afterend", afterEnd);

      Beispiel (resultierende HTML-Struktur):

      <!-- beforebegin --> <div style="color: blue;">👉 beforebegin</div> <!-- ursprüngliches Ziel-Element --> <div id="box"> <!-- afterbegin --> <div style="color: green;">👉 afterbegin</div> <!-- Originalinhalt --> <p>Inhalt der Box</p> <!-- beforeend --> <div style="color: orange;">👉 beforeend</div> </div> <!-- afterend --> <div style="color: red;">👉 afterend</div>

  • Wie löscht man ein HTML Element?

    • Ein HTML-Element kann man mit JavaScript auf zwei einfache Arten löschen:

    • element.remove(): Ist die direkteste Methode.

      • Beispiel:

      const wegElement = document.getElementById("wegDamit"); wegElement.remove();

    • parent.removeChild(child)

      • Beispiel:

      function eraseAbsatz() { const eltern = document.getElementById("container"); const kind = document.getElementById("text"); eltern.removeChild(kind); }

  • Wie kann man alle HTML Elemente eines bestimmten HTML-Elments löschen?

    • Gängigsten Methoden:

    • element.innerHTML = "" -> Entfernt alle Inhalte und Kindelemente des Containers. Schnell, aber löscht alles, auch Texte und Events.

      • Beispiel:

      const container = document.getElementById("meinContainer"); container.innerHTML = "";

    • Mit einer Schleife (while): Löscht ein Kind nach dem anderen, bis nichts mehr übrig ist.

      • Beispiel:

      const container = document.getElementById("meinContainer"); while (container.firstChild) { container.removeChild(container.firstChild); }

JavaScript (Objects)

  • Was ist ein Object in JavaScript?

    • Ein Object ist eine Sammlung von zusammengehörigen Daten und/oder Funktionen, die in sogenannten Eigenschaft–Wert-Paaren (Property–Value Pairs) gespeichert sind.

  • Wie erstellt man ein neues Object?

    • Beispiel:

    const hund = { name: "Balu", alter: 5, rasse: "Labrador", bellen: function() { console.log("Wuff!"); } };

    Erklärung:

    • name, alter, rasse und bellen sind Eigenschaften (Properties).
    • "Balu", 5 und "Labrador" und function() sind die Werte (Values).
    • bellen ist auch eine Methode (Function innerhalb eines Objekts), weil der Wert dieser Property eine Funktion ist.

  • Was ist eine Property in JavaScript?

    • Ein Schlüssel–Wert-Paar, das zu einem Objekt gehört. Eine Property beschreibt ein Merkmal eines Objekts – z. B. Name, Alter, Farbe oder auch Funktionen (Methoden).

    Beispiel:

    const hund = { name: "Luna", alter: 3 };

    Erklärung:

    • name ist eine Property mit dem Wert "Luna".
    • alter ist eine Property mit dem Wert 3.

    Allgemeine Struktur:

    const objekt = { schlüssel (key): wert(value) };

  • Auf Properties zugreifen:

    • Beispiel:

    console.log(hund.name); // "Luna" console.log(hund["alter"]); // 3

    Beide varianten sind erlaubt:

    • Punktnotation (objekt.schlüssel)
    • Klammernotation (objekt["schlüssel"])

    Die Klammernotation ist nützlich, wenn der Schlüssel dynamisch ist.

    Beispiel mit Punktnotation:

    const hund = { name: "Balu", rasse: "Labrador", alter: 6 }; console.log(hund.name); // → "Balu" console.log(hund.rasse); // → "Labrador" console.log(hund.alter); // → 6

    Beispiel mit Klammernotation (["key"]):

    console.log(hund["name"]); // → "Balu" console.log(hund["rasse"]); // → "Labrador"

  • Wie kann man Attributen auflisten?

    • Wenn man alle Properties (manchmal fälschlich „Attribute“ genannt) eines JavaScript-Objekts auflisten möchte, gibt es mehrere Methoden – je nachdem, was man genau braucht:

    • Object.keys(obj) → Alle eigenen, enumerierbaren Property-Namen

      • const hund = { name: "Balu", alter: 4, rasse: "Labrador" }; console.log(Object.keys(hund)); // → ["name", "alter", "rasse"]

    • Object.values(obj) → Nur die Werte

      • console.log(Object.values(hund)); // → ["Balu", 4, "Labrador"]

    • Object.entries(obj) → Schlüssel–Wert-Paare

      • console.log(Object.entries(hund)); // → [["name", "Balu"], ["alter", 4], ["rasse", "Labrador"]]

      Das ist ideal für Schleifen:

      const hund = { name: "Balu", alter: 4, rasse: "Labrador" }; // Mit Object.entries() und einer for...of Schleife for (const [key, value] of Object.entries(hund)) { console.log(`${key}: ${value}`); }

      Ausgabe:

      name: Balu alter: 4 rasse: Labrador

    • Mit for...in Schleife

      • Beispiel:

      const canis = { name: "Niko", alter: 2, rasse: "Rottweiler" }; // Mit for...in Schleife for (let key in canis) { console.log(key + ": " + canis[key]); }

  • Auf Properties zugreifen:

    • Es gibt verschiedene Möglichkeiten, einem bestehenden Objekt in JavaScript neue Properties hinzuzufügen.

    • Punktnotation (.)

      • Beispiel:

      const hund = { name: "Balu", alter: 4 }; // Neue Property hinzufügen hund.rasse = "Labrador"; console.log(hund); // → { name: "Balu", alter: 4, rasse: "Labrador" }

    • Klammernotation ([])

      • Beispiel:

      const hund = { name: "Balu", alter: 4 }; // Neue Property hinzufügen hund["rasse"] = "Labrador"; console.log(hund); // → { name: "Balu", alter: 4, rasse: "Labrador" }

    • Mit Object.assign()

      • Beispiel:

      const hund = { name: "Balu", alter: 4 }; // Mehrere Properties hinzufügen Object.assign(hund, { rasse: "Labrador", gewicht: 25 }); console.log(hund); // → { name: "Balu", alter: 4, rasse: "Labrador", gewicht: 25 }

      Object.assign() ist besonders nützlich, wenn man mehrere Properties gleichzeitig zu einem Objekt hinzufügen möchte.

  • Funktionen als Property:

    • In JavaScript kann eine Funktion als Property (oder „Attribut“) eines Objekts gespeichert werden. Das bedeutet, dass man einer Property des Objekts eine Funktion zuweisen kann, und diese Funktion wird dann wie jede andere Property behandelt.

    Beispiel:

    const hund = { name: "Balu", alter: 4, // Funktion als Property bellen: function() { console.log("Wuff! Wuff!"); } }; // Die Funktion aufrufen hund.bellen(); // → "Wuff! Wuff!"Was passiert wenn ich eine for...in Schleife auf eine Array anwende?

  • Anwendung der for...in Schleife auf ein Array innerhalb eines Objekts:

    • Beispiel:

    const auto = { marke: "Volkswagen", modell: "Golf", jahr: 2020, farben: ["Rot", "Blau", "Schwarz"] }; for (let index in auto.farben) { console.log(`Farbe ${index}: ${auto.farben[index]}`); }

    Ausgabe:

    Farbe 0: Rot Farbe 1: Blau Farbe 2: Schwarz

    Hier wird die for...in Schleife auf das Array auto.farben angewendet, das drei Farben enthält.

    Die Schleife iteriert über die Indizes des Arrays (0, 1, 2) und gibt den entsprechenden Wert für jede Farbe aus.
