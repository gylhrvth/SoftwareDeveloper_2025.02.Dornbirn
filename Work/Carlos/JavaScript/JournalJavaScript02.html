Mittwoch, 19. März 2025

JavaScript (Functions)

  • Was ist ein Argument und wie unterscheidet er sich von einem Parameter?

    • Argument ist der tatsächliche Wert, den man beim Funktionsaufruf übergibt. Parameter ist der Platzhalter in der Funktionsdefinition.

    function begruessung(name) { // "name" ist der Parameter console.log("Hallo, " + name + "!"); } begruessung("Paul"); // "Paul" ist das Argument

  • Was ist ein Rückgabewert? Was muss ich da berücksichtigen?

    • Ein Rückgabewert ist der Wert, den eine Funktion zurückgibt, nachdem sie ausgeführt wurde. In JavaScript (und vielen anderen Programmiersprachen) wird ein Rückgabewert mit return festgelegt.

    Falls return nicht verwendet wird, gibt die Funktion automatisch undefined zurück.

    Nach return wird die Funktion sofort beendet – Code danach wird nicht mehr ausgeführt.

    function sagHallo() { return "Hallo!"; } console.log(sagHallo()); // "Hallo!"

  • Was ist ein Objekt in JavaScript?

    • Eine Sammlung von Eigenschaften (auch Schlüssel-Wert-Paare genannt), die zusammen eine strukturierte Information darstellen. Jedes Objekt besteht aus einem Schlüssel (Key) und einem Wert (Value).

    let person = { name: "Carlos", alter: 35, beruf: "Softwareentwickler", istVerheiratet: false };

    In dem Beispiel, person ist das Objekt und hat vier Eigenschaften:

    • name ist der Schlüssel, und "Carlos" ist der Wert.
    • alter ist der Schlüssel, und 35 ist der Wert.
    • beruf ist der Schlüssel, und "Softwareentwickler" ist der Wert.
    • istVerheiratet ist der Schlüssel, und false ist der Wert.

  • Was sind Arrays in JavaScript?

    • Ein Array ist eine geordnete Sammlung von Werten, die unter einem einzelnen Variablennamen gespeichert werden. Arrays können mehrere Werte eines beliebigen Datentyps (Zahlen, Strings, Objekte, andere Arrays, etc.) enthalten und bieten eine einfache Möglichkeit, diese Werte zu organisieren und darauf zuzugreifen.

    Arrays sind nützlich, wenn man mehrere ähnliche Elemente (z.B. Zahlen oder Namen) speichern möchte.

    let zahlen = [1, 2, 3, 4, 5]; let namen = ["Carlos", "Anna", "John"];

  • Was bedeuten die Begriffe "pass by value" und "pass by reference"?

    • Pass by Value (Wertübergabe): Gilt für primitive Datentypen (number, string, boolean, null, undefined, symbol, bigint).

      • Bei einer Wertübergabe wird eine Kopie des Werts übergeben. Änderungen innerhalb der Funktion beeinflussen nicht die ursprüngliche Variable.

      function verdoppeln(x) { x = x * 2; console.log("Innerhalb der Funktion:", x); // 20 } let zahl = 10; verdoppeln(zahl); console.log("Außerhalb der Funktion:", zahl); // 10 (unverändert!)

    • Pass by Reference (Referenzübergabe): Gilt für Objekte und Arrays. Bei einer Referenzübergabe wird die Speicheradresse des Objekts übergeben, nicht eine Kopie. Änderungen in der Funktion wirken sich direkt auf das ursprüngliche Objekt aus.

      • function updatePerson(person) { person.name = "Carlos"; // Schritt 2 } let personObj = { name: "Anna" }; // Schritt 1 updatePerson(personObj); // Schritt 3 console.log(personObj.name); // Schritt 4

      Erklärung:

      • Schritt 1: Ein Objekt wird erstellt -> Hier wird ein Objekt personObj erstellt, das ein Eigenschaft name mit dem Wert "Anna" enthält.
        Das Objekt personObj ist im Speicher gespeichert, und es wird eine Referenz auf diesen Speicherort in der Variable personObj abgelegt.

      • Schritt 2: Definition der Funktion updatePerson -> Die Funktion updatePerson erwartet ein Objekt (person) als Parameter.
        Innerhalb der Funktion wird die Eigenschaft name des Objekts person auf den Wert "Carlos" gesetzt.

      • Schritt 3: Aufruf der Funktion updatePerson -> Die Funktion updatePerson wird mit personObj als Argument aufgerufen. Da Objekte in JavaScript per Referenz übergeben werden, wird nicht eine Kopie des Objekts, sondern die Referenz auf personObj an die Funktion übergeben.
        Das bedeutet, die Funktion ändert nicht das gesamte Objekt, sondern nur die Eigenschaft name des Objekts, auf das die Referenz zeigt.

      • Schritt 4: Das Ergebnis ausgeben -> Nach dem Aufruf der Funktion updatePerson(personObj) wurde die Eigenschaft name des Objekts personObj von "Anna" auf "Carlos" geändert.
        Deshalb zeigt console.log(personObj.name) den Wert "Carlos" an, da das ursprüngliche Objekt durch die Funktionsänderung tatsächlich verändert wurde.

  • Wie können Parameter durch Referenz oder Wert übergeben werden?

    • In JavaScript gibt es den Unterschied zwischen primitiven Typen (z.B. Zahlen, Strings) und komplexen Typen (z.B. Objekte, Arrays). Bei primitiven Typen wird der Wert kopiert, während bei komplexen Typen die Referenz übergeben wird.

  • Wie funktioniert die Übergabe einer Funktion in JavaScript?

    • In JavaScript kann man eine Funktion als Argument an eine andere Funktion übergeben. Das ist möglich, weil Funktionen in JavaScript Objekte sind und wie jede andere Variable behandelt werden können.

    Beispiel:

    function greeting(name) { console.log("Hello, " + name); } function execute(func, name) { func(name); // The passed function is called here } execute(greeting, "Carlos"); // "Hello, Carlos"

    Erklärung:

    1. The greeting function prints a greeting message when given a name.
    2. The execute function expects a function and a name as arguments. It calls the passed function with the name. (Inside execute, func(name) calls the function that was passed (e.g., greeting) and provides name as an argument to it).
    3. When execute(greeting, "Carlos") is called, it calls the greeting function with "Carlos" as the argument, and the greeting is displayed in the console.

Dienstag, 18. März 2025

JavaScript (Functions)

  • Was sind Funktionen? Wofür sind sie nützlich?

    • Funktionen in JavaScript (und in anderen Programmiersprachen) sind wiederverwendbare Codeblöcke, die eine bestimmte Aufgabe ausführen. Sie erlauben es, einen vordefinierten Satz von Anweisungen mit einem Namen zu versehen und diesen bei Bedarf immer wieder aufzurufen, anstatt den gleichen Code mehrfach zu schreiben.

    -> Wofür sind Funktionen nützlich?:

    • Wiederverwendbarkeit

    • Bessere Strukturierung

    • Lesbarkeit und Wartbarkeit

    • Fehlervermeidung

    • Eingabewerte und Flexibilität

  • Was sind eingabe Parametern einer Funktion?

    • Eingabeparameter (auch Argumente genannt) sind Werte, die einer Funktion beim Aufruf übergeben werden. Eine Funktion kann diese Werte verwenden, um Berechnungen durchzuführen oder eine bestimmte Aktion auszuführen.

    function nameDerFunktion(parameter1, parameter2) { // Die Funktion verwendet parameter1 und parameter2 }

    parameter1, parameter2 → Platzhalter für Werte, die beim Funktionsaufruf übergeben werden.

    Die Funktion kann diese Parameter innerhalb ihres Codes verwenden.

    Beispiel:

    function greet(name){ console.log("Hi, " + name + "!"); } greet("Carlos"); // "Hi, Carlos!"

  • Wie kann man Parameter als Optional verwenden?

    • So kann man Standardwerte verwenden:

    function greet(name = "Gast") { console.log("Hallo, " + name + "!"); } greet(); // "Hallo, Gast!" (kein Parameter übergeben) greet("Carlos"); // "Hallo, Carlos!"
