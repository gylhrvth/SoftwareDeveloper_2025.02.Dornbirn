SQL (Node JS & Express Setup)

Steps:

  • Step 00: NodeJS auf dem Rechner installieren.

  • Step 01: In db-ejs / db-rest-api (Gyulas Beispiel Projekte), ein '.env' File erstellen.

  • Step 02: In dem .env File, folgende Struktur reingeben:

    SQL_SERVER=localhost
    SQL_USER= User ('Carlos' in diesem Fall)
    SQL_PASSWORD= Password ('1103' in diesem Fall)
    SQL_DATABASE=mondial (oder meine gewünschte DB)
    HTTP_PORT=3000 (Kann auch 3001 sein)

  • Step 03: In VSCode, Terminal beim aktuellen Project öffnen.

  • Step 04: Terminal Befehl -> npm install

  • Step 04: Terminal Befehl -> npm run dev (beim Erfolg, Message 'Example app listening on port 3000' wird gezeigt)

  • Auf dem Browser, http://localhost:3000/ eingeben
