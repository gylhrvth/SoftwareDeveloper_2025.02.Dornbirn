    04. März 2025

    CSS Transform:

  • Was ist ein transform Attribute?

    • Das transform-Attribut in CSS verändert das Erscheinungsbild eines Elements, ohne das Layout zu beeinflussen.

    Es Ermöglicht:

    1. Verschieben: transform: translate(5opx, 100px)
    2. Drehen: transform: rotate(45deg);
    3. Skalieren: transform: scale(1.5)
    4. Kippen: transform: skew(30deg, 20deg);
