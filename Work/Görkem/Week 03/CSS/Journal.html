CSS Transform

  • Was ist CSS Transform?

    • Es ist eine CSS-Eigenschaft, die es ermöglicht, HTML-Elemente visuell zu verändern, ohne ihre tatsächliche Position im Dokumentenfluss zu ändern. Sie wird verwendet, um Elemente zu drehen,
    skalieren, verschieben oder kippen.

  • Wie verwendet man sie, welche Eigenschaften sind üblich?

      • scale (x,y): Vergrößert oder verkleinert das Element.

      div { transform: scale(1.5, 2); }

      rotate(angle): Dreht das Element im Uhrzeigersinn um den angegebenen Winkel.

      div { transform: rotate(45deg); }

      translate (x,y): Verschiebt ein Element entlang der X- und Y-Achse.

      div { transform: translate(50px, 100px); }

      skew(x-angle, y-angle): Kippt das Element entlang der X- oder Y-Achse.

      div { transform: skew(20deg, 10deg); }

    CSS Transition

  • Was ist eine CSS Transition?

    • CSS Transitions ermöglichen eine sanfte, animierte Änderung von CSS-Eigenschaften. Sie verhindern abrupte Änderungen und lassen z. B. Farb-, Größen- oder Positionsänderungen fließend ablaufen.


  • Wie kann man CSS Transitions verwenden?

    • Syntax in CSS:

    transition: property duration timing-function delay;

    • duration → Wie lange die Animation dauert (z. B. 0.5s, 1s).

    • timing-function (optional) → Der Animationsverlauf (ease, linear, ease-in-out, etc.).

    • delay (optional) → Verzögerung vor Start der Animation.

    • property → Die zu animierende Eigenschaft (z. B. background-color, opacity, transform).

  • Was sind die Timing Funktionen, welche kennst du?

    • Die timing-function in CSS bestimmt, wie die Geschwindigkeit einer Transition oder Animation über die Zeit verläuft, z.B. gleichmäßig oder mit bestimmten Mustern.

    Hier sind einige wichtige Timing-Function-Werte in CSS:

      ✅ ease: Beginnt langsam, wird dann schneller und endet langsam

      ✅ ease-in: Beginnt langsam und beschleunigt sich

      ✅ ease-out: Beginnt schnell und verlangsamt sich am Ende

      ✅ ease-in-out: Beginnt und endet langsam, wird in der Mitte schneller

      ✅ linear: Die Änderung erfolgt gleichmäßig über die gesamte Zeit
