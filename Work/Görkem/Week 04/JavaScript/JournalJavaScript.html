JavaScript Basics

  • Was ist ein Java Script?

    JavaScript ist eine dynamische Programmiersprache, die hauptsächlich in der Webentwicklung verwendet wird.

    Beispiel:

    alert("Hallo,Welt");
  • Was kann man mit Java Script in verbindung mit Web-Development machen?

    JavaScript wird häufig in der Webentwicklung verwendet, um interaktive und dynamische Websites zu erstellen. Hier sind einige Anwendungsfälle:

    • Erstellen von interaktiven Benutzeroberflächen (UI)
    • Formularvalidierung auf der Client-Seite
    • Dynamisches Nachladen von Inhalten ohne die Seite neu zu laden (AJAX)
    • Erstellen von Animationen und Übergängen
    • Erstellen von Webanwendungen mit Frameworks wie React, Angular, oder Vue.js
    • Arbeiten mit APIs (Geolocation, Web Storage, etc.)
  • Welche Möglichkeiten gibt es, JavaScript-Code zu schreiben und in ein Projekt einzufügen?

    Es gibt verschiedene Möglichkeiten, JavaScript in ein Projekt zu integrieren:

    • Inline Script: Direkt im HTML-Tag einfügen.
      • <button onclick="alert('Hallo, Welt!')">Klicke mich</button>

    • Im <script>-Tag innerhalb von HTML

      • <script> console.log("Hallo, Welt!"); </script>

    • Externe JavaScript-Datei: Externe JavaScript-Datei (.js-Datei)

      • <script src="script.js"></script>

    • Modulbasiertes System mit type="module"

      • <script type="module" src="script.js"></script>

      JavaScript im Server-Umfeld mit Node.js

  • Wie setzt man Kommentare in JavaScript?

    • In JavaScript gibt es zwei Kommentararten:

    • Einzeiliger Kommentar (//)

      
            // Das ist ein einzeiliger Kommentar
            console.log("Hallo, Welt!"); // Kommentar nach einer Codezeile

    • Mehrzeiliger Kommentar (/* ... */)

      
            /*
            Das ist ein mehrzeiliger Kommentar.
            Er kann über mehrere Zeilen gehen.
            *\
            console.log("Hallo, Welt!");

  • Wenn man Namen definiert für Variablen oder Funktionen, welche Regeln muss man beachten?

    • ✅ Erlaubt:

      • Buchstaben (a-z, A-Z)

      • Ziffern (0-9), aber nicht am Anfang

      • _ (Unterstrich) und $ (Dollarzeichen)

      • Groß- und Kleinschreibung wird unterschieden (name ≠ Name)

    • ❌ Nicht erlaubt:

      • Namen dürfen nicht mit einer Zahl beginnen (1variable ❌)

      • Keine Leerzeichen oder Sonderzeichen außer _ und $

      • Reservierte Wörter (z. B. let, function, return) dürfen nicht als Namen verwendet werden


    JS Programming Basics

    • Was sind sogenannte Statements?

      • Ein Statement (Anweisung) in JavaScript ist eine einzelne ausführbare Codezeile, die eine bestimmte Aktion ausführt. (z. B. eine Variable setzen, eine Bedingung prüfen oder eine Schleife ausführen).

      let name = "Welt"; // Variablenzuweisung console.log(name); // Funktionsaufruf if (name === "Welt") { // Bedingung console.log("Hallo, Welt!"); }

    • Was ist Syntax?

      • Syntax bezeichnet die Regeln und Struktur einer Programmiersprache, die festlegen, wie Code korrekt geschrieben werden muss, damit der Computer ihn versteht und ausführt.

    • Was sind Variablen und wie werden sie in JavaScript definiert?

      • Variablen sind Platzhalter für Daten in einem Programm. Sie ermöglichen es, Werte zu speichern und später im Code zu verwenden. In JavaScript sind Variablen flexible Behälter, die unterschiedliche Datentypen aufnehmen können (z. B. Zahlen, Texte, Wahrheitswerte).

      Werte können unterschiedlich sein, z. B. Zahlen (30), Strings ("Hallo"), Booleans (true/false), Arrays oder Objekte.

      let name = "Ahmet"; // Variablenname: name, Wert: "Ahmet" let age = 35; // Variablenname: age, Wert: 35

    • Was sind Booleans?

      • Booleans sind ein Datentyp in JavaScript, der nur zwei mögliche Werte haben kann: true (wahr) oder false (falsch). Sie werden häufig in Bedingungen, Vergleichen und logischen Operationen verwendet.

    • Was sind Operatoren in JavaScript?

      • Operatoren sind spezielle Zeichen, die bestimmte Aktionen wie Berechnungen, Vergleiche oder logische Operationen auf Variablen oder Werten ausführen.

    • Was ist eine Funktion in JavaScript?

      • Eine Funktion in JavaScript ist ein wiederverwendbarer Codeblock, der eine bestimmte Aufgabe oder Berechnung ausführt. Funktionen ermöglichen es, komplexe Aufgaben in kleinere, handhabbare Teile zu unterteilen und den Code effizienter und übersichtlicher zu gestalten.

      function greet() { console.log("Hallo, Ahmet!"); } greet(); // Ruft die Funktion auf und gibt "Hallo, Ahmet!" aus

    • Was ist eine Schleife (Loop)?

      • Eine Schleife in JavaScript ist eine Kontrollstruktur, die es ermöglicht, einen bestimmten Codeblock wiederholt auszuführen, solange eine Bedingung erfüllt ist. Schleifen sind nützlich, wenn man eine Aktion mehrmals durchführen möchte, ohne den gleichen Code mehrfach zu schreiben.

      Es gibt verschiedene Arten von Schleifen (z. B. for, while, do...while), die sich durch ihre Struktur und die Art der Bedingungsprüfung unterscheiden.

    • Was ist das If-Else Statement?

      • Das if-else-Statement in JavaScript ist eine Bedingungsanweisung, die es ermöglicht, unterschiedliche Codeblöcke auszuführen, je nachdem, ob eine bestimmte Bedingung wahr (true) oder falsch (false) ist.

    JavaScript Datatypes

  • Was sind die 7 + 1 standard JS Types?

    • In JavaScript gibt es insgesamt 8 grundlegende Datentypen. Sie werden oft als "7 + 1" bezeichnet, weil es 7 traditionelle Datentypen gibt und der 8. Datentyp als neuer hinzukam.

    Die 7 traditionellen Datentypen:

    1. Number: Repräsentiert Zahlen, sowohl Ganzzahlen als auch Dezimalzahlen.
    2. String: Repräsentiert Text.
    3. Boolean: Repräsentiert Wahrheitswerte (true oder false).
    4. Object: Repräsentiert Sammlungen von Daten oder komplexe Entitäten.
    5. Undefined: Ein Wert, der einer Variablen zugewiesen wird, die nicht initialisiert wurde.
    6. Null: Ein spezieller Wert, der "leer" oder "nicht vorhanden" bedeutet.
    7. Symbol: Ein neuer Datentyp für einzigartige, unveränderliche Identifikatoren.

    Der zusätzliche 8. Datentyp:

    • BigInt: Ein Datentyp für sehr große Ganzzahlen, die über den Bereich des Number-Typs hinausgehen.

    Jeder dieser Datentypen erfüllt eine spezielle Aufgabe und wird in unterschiedlichen Szenarien verwendet. Besonders BigInt wurde hinzugefügt, um die Bedürfnisse für große Zahlen zu decken, die in der Vergangenheit zu Fehlern oder Ungenauigkeiten führten.

  • Was passiert, wenn ich 10 + "Apfel" schreibe? Warum?

    Wenn man 10 + "Apfel" schreibt, wird die Zahl 10 automatisch in einen String umgewandelt, da der + - Operator in JavaScript auch zur Verkettung von Strings dient.

    Das Ergebnis dieser Operation ist der String: "10Apfel"

    console.log(10 + "Apfel"); // Ausgabe: "10Apfel"

  • Was passiert wenn ich 0.1 + 0.2 schreibe? Warum?

    Wenn man 0.1 + 0.2 schreibt, ergibt das 0.30000000000000004 und nicht exakt 0.3, aufgrund der Art und Weise, wie Fließkommazahlen in JavaScript im Binärformat gespeichert werden.

    console.log(0.1 + 0.2); // Ausgabe: 0.30000000000000004

  • Was macht die toString() Funktion?

    Die toString() Funktion wird verwendet, um Werte in eine String-Darstellung umzuwandeln. Sie ist besonders geeignet, wenn du mit primitiven Datentypen wie Zahlen oder Booleans als Strings arbeiten möchtest, sie ist auch nützlich, um Daten wie Datum und Uhrzeit als String darzustellen.

    let number = 123; console.log(number.toString()); // Ausgabe: "123" let booleanValue = true; console.log(booleanValue.toString()); // Ausgabe: "true" let arr = [1, 2, 3]; console.log(arr.toString()); // Ausgabe: "1,2,3"

  • Was macht die valueOf() Funktion? Wofür ist die geeignet?

    Die valueOf()-Methode in JavaScript gibt den primitiven Wert eines Objekts zurück. Sie wird in der Regel intern verwendet, um den Wert eines Objekts in Berechnungen oder Vergleichen zu nutzen, wenn der Operator wie +, -, * oder == aufgerufen wird.

    let numObj = new Number(10); console.log(numObj.valueOf()); // Ausgabe: 10

  • Was ist der typeof Operator? Wofür ist der nützlich?

    Der typeof-Operator in JavaScript gibt den Datentyp eines Wertes oder Ausdrucks zurück. Es handelt sich um einen Unary-Operator, der den Typ eines Operanden als String zurückgibt. Ist nützlich für die Fehlervermeidung durch Überprüfung des Datentyps vor der Weiterverarbeitung von Werten.

    let num = 42; console.log(typeof num); // Ausgabe: "number"
