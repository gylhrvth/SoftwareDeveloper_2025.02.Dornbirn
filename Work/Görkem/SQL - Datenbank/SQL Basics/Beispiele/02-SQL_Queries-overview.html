  1. Wie viele Einwohner hat Österreich?

    2. SELECT Name, Population
    FROM country
    WHERE Name = 'Austria';

    Lösung: Die Abfrage liefert die Einwohnerzahl von Austria (Einwohnerzahl: 8.023.244)

  2. Wie viele Einwohner hat der Kontinent Europa?

    3. SELECT
    SUM(country.Population) AS Total_population
    FROM country
    JOIN encompasses ON country.code = encompasses.Country
    WHERE encompasses.Continent = 'Europe';

    Lösung: Das Ergebnis von 791.243.404 ist die Gesamtbevölkerung der Länder, die in Europa liegen.

  5. Wie viel Prozent der Menschen leben im Kontinent Europa?

    6. SELECT
    (europe.Population / world.TotalPopulation) * 100 AS PercentageInEurope
    FROM (SELECT SUM(country.Population)
    FROM country
    JOIN encompasses ON country.Code = encompasses.Country
    WHERE encompasses.Continent = 'Europe') AS europe,
    (SELECT SUM(Population) AS TotalPopulation FROM country) AS world;

    Lösung: 13,70%
