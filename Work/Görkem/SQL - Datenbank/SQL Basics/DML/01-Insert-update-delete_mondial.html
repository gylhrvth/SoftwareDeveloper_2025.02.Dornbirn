SQL: Insert, Update & Delete

  1. Erstelle eine neue Stadt in der Mondial Datenbank

    2. INSERT INTO city (name, country, Province, Population, Latitude, Longitude)
    VALUES ('Sinop', 'TR', 'Sinop', 58000, 42.0268, 35.1511);

    Lösung:


    Name Country Province Population Latitude Longitude
    Sinop TR Sinop 58000 42.0268 35.1511

    Diese Abfrage aktualisiert die folgenden Werte der Stadt Sinop.

    UPDATE city
    SET population = 58000,
    longitude = 35.1511,
    latitude = 42.0268
    WHERE name = 'Sinop';


    Diese Abfrage wählt alle Spalten aus der Tabelle 'city' aus, bei denen der Städtename 'Sinop' ist (aktualisiere zu Esenler).

    SELECT *
    FROM city
    WHERE name = 'Esenler';


  2. Stadt umbennenen (UPDATE city)

    3. UPDATE city
    SET name = 'Esenler'
    WHERE name = 'Sinop';

    Lösung:


    Name Country Province Population Latitude Longitude
    Esenler TR Sinop 58000 42.0268 35.1511

  3. Lösche die von dir erstellte Stadt (DELETE city)

    4. DELETE FROM city
    WHERE name = 'Esenler'
    AND country = 'TR'
    AND province = 'Sinop';

    Überprüft dannach ob die Daten aus der Tabelle gelöscht wurde

    SELECT * FROM city WHERE name = 'Esenler';



  1. Erstelle eine neue Berg

    2. INSERT INTO mountain (name, mountains, height, latitude, longitude)
    VALUES ('Mount Ararat', 'TR', 5137, 39.7020, 44.3117);

    Lösung:


    Name Country Height (m) Latitude Longitude
    Mount Ararat TR 5137 39.7020 44.3117

    Diese Abfrage wählt alle Spalten aus der Tabelle 'mountain' aus, bei denen der Bergname 'Mount Ararat' ist.

    SELECT *
    FROM mountain
    WHERE name = 'Mount Ararat';


  2. Berg vergrößern (UPDATE mountain)

    3. UPDATE mountain
    SET height = 6000
    WHERE name = 'Mount Ararat';

    Lösung:


    Name Country Height (m) Latitude Longitude
    Mount Ararat TR 6000 39.7020 44.3117

  3. Lösche die von dir erstellte Berg (DELETE mountain)

    4. DELETE FROM mountain
    WHERE name = 'Mount Ararat'
    AND mountains = 'TR';

    Überprüft dannach ob die Daten aus der Tabelle gelöscht wurde

    SELECT * FROM mountain
    WHERE name = 'Mount Ararat';
