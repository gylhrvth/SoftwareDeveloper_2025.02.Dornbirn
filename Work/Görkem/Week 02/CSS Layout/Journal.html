    CSS Layout:

  • Welche CSS-Techniken kann man zur Gestaltung eines Layouts verwenden?

    • CSS Positioning (absolute, relative, fixed, sticky).

    CSS Frameworks (z. B. Bootstrap, Tailwind CSS).

    CSS Grid - Rasterbasiertes Layout (display: grid) → Perfekt für komplexe, flexible Layouts.

    CSS Flexbox → Gut für einzeilige oder kleinere Layouts. Flexbox ist eindimensional: entweder in einer Reihe oder in einer Spalte.

    CSS Floats → Veraltet, aber immer noch in einigen Projekten verwendet.CSS Float (veraltet, aber noch in manchen Fällen genutzt):
    Wenn man Text um ein Bild oder ein anderes Element fließen lassen möchte, kann man float verwenden. Das Bild wird “links” oder “rechts” positioniert.

  • Wie erstellt man ein Flexbox layout?

    • Ein Flexbox Layout wird mit der CSS-Eigenschaft display: flex erstellt. Flexbox ermöglicht eine einfache und flexible Möglichkeit, Elemente in einer Reihe oder Spalte anzuordnen und auszurichten.

    🚀Grundstruktur

    1. Container: Das Elternelement, das die Flexbox enthält. Mit display: flex; wird die Flexbox erstellt.
    2. Flex-Items: Die Elemente innerhalb des Containers, die in einer Reihe oder Spalte angeordnet werden.

  • Flexbox Eigenschaften

    • 🚀Container Eigenschaften:

      display: flex -> Aktiviert das Flexbox-Modell für den Container und macht alle direkten Kinder zu Flex-Items.

      flex-direction: Bestimmt die Ausrichtung der Flex-Items entlang der Hauptachse.
      Werte: row / column / row-reverse / column-reverse

      flex-wrap: Bestimmt, ob Flex-Items in mehreren Zeilen oder Spalten umgebrochen werden können, wenn der Platz nicht ausreicht.
      Werte: wrap / wrap-reverse / nowrap

      justify-content: Steuert die horizontale Ausrichtung der Flex-Items entlang der Hauptachse.
      Werte: flex-start / flex-end / center / space-between / space-around / space-evenly

      align-items: Steuert die vertikale Ausrichtung der Flex-Items entlang der Querachse.
      Werte: flex-start / flex-end / center / baseline / stretch

      align-content: Steuert den Abstand zwischen mehreren Flexbox-Zeilen (nur bei flex-wrap!!!).
      Werte: flex-start / flex-end / center / space-between / space-around / stretch


    🚀Child-Item Eigenschaften:

      order: Bestimmt die Reihenfolge der Flex-Items.

      flex-grow: Bestimmt, wie viel Platz ein Flex-Item in der Flexbox einnehmen kann.

      flex-shrink: Bestimmt, wie viel Platz ein Flex-Item in der Flexbox abgeben kann.

      flex-basis: Bestimmt die Größe eines Flex-Items, bevor der freie Platz aufgeteilt wird.

      flex: Kurzschreibweise für flex-grow, flex-shrink und flex-basis.

      align-self: Überschreibt die align-items-Eigenschaft des Containers für ein einzelnes Flex-Item.

  • Wie erstellt man ein Grid layout?

    1. Spalten und Zeilen definieren: Festlegen, wie viele Spalten und Zeilen man im Grid haben möchte, Eigenschaften grid-template-columns und grid-template-rows.

    2. Grid-Elemente platzieren: Grid-Items anordnen: Die direkten Kinder des Containers (die Grid-Items) werden automatisch in den Grid-Zellen platziert.

    3. Container definieren: Der Container, der das Grid enthält, wird mit display: grid aktiviert.


    🚀Grid-Code Beispiel

    html:

    <div class="container"> <div class="item">Item 1</div> <div class="item">Item 2</div> <div class="item">Item 3</div> <div class="item">Item 4</div> <div class="item">Item 5</div> <div class="item">Item 6</div> </div>

    CSS:

    .container { display: grid; grid-template-columns: 2fr 1fr 2fr; grid-template-rows: 120px auto 120px; grid-gap: 10px; } .item { background-color: blue; padding: 10px; text-align: center; }

    🚀Erklärung

    Die erste und dritte Spalte sind gleich breit (2fr), und die mittlere Spalte ist doppelt so breit (1fr).

    Es ist so definiert, dass die erste Die Zeilen sind so definiert, dass die erste und letzte Zeile jeweils 120px hoch sind, und die mittlere Zeile nimmt den verbleibenden Platz (auto: nimmt nur so viel Platz wie der Inhalt braucht).
