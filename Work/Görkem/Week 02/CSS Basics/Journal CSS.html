    Was ist CSS?

    CSS (Cascading Style Sheets) ist eine Stylesheet-Sprache, die das **Design und Layout** von HTML-Webseiten bestimmt.

    Wofür wird CSS verwendet?

    • 🎨 Farben & Hintergründe
    • 📝 Schriftarten & Textgestaltung
    • 📏 Abstände & Layout (Flexbox, Grid)
    • ✨ Animationen & Effekte
    • 📱 Responsive Design (Mobilfreundlichkeit)

    Kurz: CSS macht Webseiten schön, strukturiert und interaktiv! 🚀


    Was für möglichkeiten gibt es CSS in einem Dokument zu verwenden?

    Externe CSS (.css) ✅ Empfohlen für sauberen, wiederverwendbaren Code


    Internes CSS (<style>) ✅ Gut für kleine Projekte oder einzelne Seiten


    Inline-CSS (direkt im HTML-Tag) ❌ Nur in Ausnahmefällen nutzen.

    💡 Beste Wahl: Externe CSS-Datei für Struktur & Wartbarkeit! 🚀


    Was ist der Unterschied, Vor- und Nachteile von den 3 verschiedenen möglichkeiten?

    Externe CSS
    ✅ Sauberer, wiederverwendbarer Code
    ✅ Trennung von Struktur (HTML) und Design (CSS)
    ❌ Zusätzliche Datei muss geladen werden
    ❌ Kein direkter Einfluss auf einzelne Elemente


    Internes CSS

    ✅ Schnell und einfach für kleine Projekte
    ❌ Design und Struktur in einem Dokument
    ❌ Keine Trennung von HTML und CSS


    Inline-CSS

    ✅ Direkter Einfluss auf einzelne Elemente
    ❌ Unübersichtlich und schwer wartbar
    ❌ Design und Struktur in einem Dokument


    💡Fazit:

    ✅ Externe CSS-Dateien sind die beste Wahl für größere Projekte!

    ✅ Internes CSS ist gut für kleine Seiten oder Tests.

    ❌ Inline-CSS sollte nur in Ausnahmefällen genutzt werden.


    Wie bindet man eine CSS-Datei in ein HTML-Dokument ein?

    📌 Schritte zur Anwendung:

    1. Projektordner anlegen: Ich erstelle einen Ordner z. B. Journal/.
    2. Wichtige Dateien: Ich erstelle:
      • index.html (für die Struktur)
      • css/style.css (für das Design)
    3. CSS einbinden: Ich verknüpfe die Datei mit: 
      <link rel="stylesheet" href="css/style.css">
    4. VS Code Tools: Ich nutze Live Server für eine Vorschau und Prettier für sauberen Code.

    Was sind Selektoren und wie funktionieren sie?

    Selektoren sind CSS-Regeln, die bestimmen, welche Elemente auf einer Webseite gestylt werden.
    Sie können nach Elementen, Klassen, IDs oder Attributen auswählen.
    Beispiel: h1 { color: red; } (Selektor: h1, Eigenschaft: color, Wert: red)
