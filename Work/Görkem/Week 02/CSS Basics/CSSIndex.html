Was ist CSS?

CSS (Cascading Style Sheets) ist eine Stylesheet-Sprache, die das **Design und Layout** von HTML-Webseiten bestimmt.

Wofür wird CSS verwendet?

  • 🎨 Farben & Hintergründe
  • 📝 Schriftarten & Textgestaltung
  • 📏 Abstände & Layout (Flexbox, Grid)
  • ✨ Animationen & Effekte
  • 📱 Responsive Design (Mobilfreundlichkeit)

Kurz: CSS macht Webseiten schön, strukturiert und interaktiv! 🚀
