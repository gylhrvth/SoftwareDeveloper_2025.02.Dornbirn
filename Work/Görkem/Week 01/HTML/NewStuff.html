HTML Basics:

  • Was ist ein HTML, wofür wird es verwendet?

    • HTML (HyperText Markup Language) ist die Sprache, die verwendet wird, um Website zu strukturieren.
    Sie besteht aus sogennanten Tags, Text, Bilder, Links und andere Inhalte auf einer Seite definiert.

    HTML alleine reicht nicht aus, um eine ansprechende Webseite zu gestalten oder interaktive Elemente zu erstellen.
    Dafür werden CSS (für das Design) und JavaScript (für die Funktionalität) verwendet.
