HTML Basics:

  • Was ist ein HTML, wofür wird es verwendet?

    • HTML (HyperText Markup Language) ist die Sprache, die verwendet wird, um Website zu strukturieren.
    Sie besteht aus sogennanten Tags, Text, Bilder, Links und andere Inhalte auf einer Seite definiert.

    HTML alleine reicht nicht aus, um eine ansprechende Webseite zu gestalten oder interaktive Elemente zu erstellen.
    Dafür werden CSS (für das Design) und JavaScript (für die Funktionalität) verwendet.

Hauptfunktion:

    Ihre Hauptfunktion besteht darin, die Struktur und den Inhalt einer Website zu definieren und zu strukturieren. Dazu gehören:

  • Textelemente: Überschriften -> (h1) bis (h6) / Absätze -> (p) / Listen -> (ul), (ol), (li)
  • Multimedia-Inhalte (Bilder, Videos, Audio)
  • Links und Navigation (Hyperlinks, Menüs)
  • Formulare (Eingabefelder, Buttons)
  • Semantische Struktur (z. B. (header), (article), (footer) für eine bessere Verständlichkeit)

Was sind Tags?

Tags sind die Bausteine von HTML. Sie werden verwendet, um Elemente in einer Webseite zu definieren und zu strukturieren.
Ein Tag besteht aus einem Namen, der in spitzen Klammern (< >) steht.

Arten von Tags

  • Öffnende und schließende Tags: Die meisten Tags bestehen aus einem öffnenden Tag, einem Inhalt und einem schließenden Tag.
    Beispiel: <p>Das ist ein Absatz.</p>

  • Selbstschließende Tags: Einige Tags benötigen keinen separaten Schlusstag. Sie schließen sich selbst.
    Beispiel: <img src="bild.jpg" alt="Ein Beispielbild">

  • Attribute in Tags: Tags können Attribute enthalten, die zusätzliche Informationen bereitstellen.
    Beispiel: <a href="https://example.com">Besuche die Webseite</a>

Wie fügt man Bilder in eine HTML-Seite ein?

Bilder werden in HTML mit dem Tag eingefügt. Dieser Tag ist selbstschließend, das heißt, er benötigt kein separates Schlusstag (< /img>).
Struktur: Attribute -> src (source) -> Gibt den Pfad zur Bilddatei an (lokal oder URL). / Attribute -> alt (alternative Beschreibung).
Beispiel: <img src="https://example.com/bild.jpg" alt="Ein Bild aus dem Internet">

Wichtigste HTML Bestandteile:

  • <!DOCTYPE html>: Definiert das Dokument als HTML5.
  • <html>: Umfasst das gesamte HTML-Dokument.
  • <head>: Enthält Meta-Informationen und Links zu Ressourcen.
  • <body>: Enthält den sichtbaren Inhalt der Seite. Tags wie header, nav, main, section, footer befinden sich innerhalb des body Tags.
  • <header>: Kopfbereich der Seite (z. B. Titel, Navigation).
  • <nav>: Navigationselemente (Links zu Seiten oder Abschnitten).
  • <main>: Der Hauptinhalt der Seite.
  • <section>: Thematisch zusammengehörende Inhaltsabschnitte.
  • <footer>: Fußbereich (z. B. Copyright, Kontaktinformationen).

Was sind HTML-Attribute und wie werden sie verwendet?

HTML-Attribute sind zusätzliche Informationen, die in den Tags enthalten sind. Sie geben zusätzliche Informationen über das Element und seine Funktionalität.
Sie werden in Form von Attribut-Wert-Paaren innerhalb des Tags angegeben.

Wie kann man CSS in HTML intergrieren?

  • 1. Inline-CSS: CSS wird direkt im HTML-Tag über das style-Attribute angewendet. Dies ist einfachste Methode, eignet sich
    jedoch nur für sehr kleine Anpassungen.

  • 2. Internes-CSS: Internes CSS (im <head>-Bereich): CSS kann innerhalb des <style>-Tags im <head>-Bereich der HTML-Seite eingebunden werden. Dies ermöglicht eine zentralisierte Gestaltung für die gesamte Seite.

  • 3. Externes-CSS: Externes CSS wird in einer separaten Datei gespeichert und über den <link>-Tag in das HTML-Dokument eingebunden. Dies ermöglicht eine zentrale Verwaltung des Stylesheets für mehrere Seiten.

Das display-Attribut

Das display-Attribut in CSS wird verwendet, um das Verhalten eines Elements auf einer Webseite zu steuern.
Es definiert, wie ein Element auf der Seite angezeigt wird und wie es sich zu anderen Elementen verhält.
Die häufigsten Werte für das display-Attribut sind: block, inline, inline-block, none, flex, grid.
